# inbox — входящие материалы

Сюда помещаются сырые заметки, ссылки, голосовые расшифровки, быстрые идеи и материалы, которые ещё не разобраны.

## Правило обработки

- Не хранить здесь утверждённые решения.
- Не считать материалы из inbox частью канона.
- После разбора переносить в `ideas/`, `docs/`, `references/` или `99_ARCHIVE/`.

## Формат записи

```md
# YYYY-MM-DD — краткое название

Статус: INBOX
Источник:
Суть:
Куда перенести:
Следующий шаг:
```
