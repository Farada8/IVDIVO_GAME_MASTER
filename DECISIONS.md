# DECISIONS — журнал решений основателя

Статус: ACTIVE  
Дата обновления: 2026-07-22  
Назначение: фиксировать утверждённые, отменённые и изменённые решения проекта.

## Правила

- Только основатель может присвоить статус APPROVED.
- Новая гипотеза не заменяет утверждённое решение без записи в этом файле.
- Каждое фундаментальное изменение получает ID формата `DEC-000`.
- Если решение меняет главный контекст, обновляется `PROJECT_CORE_CONTEXT.md`.
- Старые решения не удаляются: они получают статус `SUPERSEDED BY DEC-...`.

## Новые решения текущей уборки

### DEC-018 — Главный жанр: Dark Cosmic Adventure

Статус: APPROVED / FOUNDER-DECISION  
Дата фиксации: 2026-07-22  
Источник: команда основателя «убери старьё...» + ранее зафиксированный STUDIO_BRAIN / текущая разработка 2026-07-22  
Заменяет: DEC-002 как horror-primary framing

Активное жанровое определение проекта:

```text
Dark Cosmic Adventure = тёмный вход + космическая вертикаль + восхождение + преобразование, а не splatter-horror и не чистая тёмная данжен-игра.
```

Horror, страх, нижние миры и тьма остаются допустимыми эмоциональными инструментами нижних слоёв игры, но не являются главным продуктовым определением. Основная эмоция: благоговение, опасность, масштаб, выбор, восхождение, катарсис.

### DEC-019 — Архитектура: пять ярусов + орбитали

Статус: APPROVED / FOUNDER-DECISION  
Дата фиксации: 2026-07-22  
Источник: текущие founder-решения и документы `docs/06_MECHANICS/*`  
Заменяет: DEC-003, DEC-012 как конфликтные архитектурные основы

Активная архитектура:

```text
1. Физический ярус / мир
2. Тонкий ярус / мир
3. Метагалактический ярус / мир
4. Огненный ярус / переход
5. Синтезный ярус / мир
+ орбитальные / параллельные миры
```

Орбитали не равны внутренним клеткам / секторам. Сектора и клетки — это внутренние состояния круга мира. Орбитали — параллельные миры / кольца, входящие в основной мир и друг в друга.

Старая модель восьми вертикальных миров и модель 16 миров / 1024 уровней больше не используются как активный канон.

### DEC-020 — Ресурсная модель: Дух / Свет / Огонь

Статус: APPROVED / FOUNDER-DECISION / TO-BALANCE  
Дата фиксации: 2026-07-22  
Источник: RESOURCE_MODEL_v0.2_DRAFT и текущая команда очистки  
Заменяет: старый слой `Луч` как центральный / единый ресурс

Активная ресурсная рамка:

```text
Дух — ресурс волевого / метагалактического прохождения, силы пути, выдерживания и верхней направленности.
Свет — ресурс знания, раскрытия, ясности, очищения и тонкого различения.
Огонь — ресурс перехода, суда, переплавки, очищения и высокой трансформации.
```

Дополнительные уточнения:

- энергия может использоваться как сырьё / низовой ресурс;
- Синтез является целью / верхним состоянием / финальным режимом, а не обычной валютой ранней игры;
- `Луч` сохраняется как канонически-чувствительный источник / направление / школьный вектор, но не как единственная центральная валюта прототипа.

Числа ресурсов остаются `TO-BALANCE`.

### DEC-021 — Движение: кольца / кубик / секторы / аттестации

Статус: APPROVED / FOUNDER-DECISION / TO-PLAYTEST  
Дата фиксации: 2026-07-22  
Источник: MOVEMENT_MODEL_v0.1_DRAFT, текущая команда очистки

Активная рамка движения:

```text
Игроки движутся по кольцам / дискам / секторам.
Кубик используется для качества, риска, цены, сдвига, силы события или редкой конъюнкции, но не заменяет решение игрока.
Переход выше не происходит просто потому, что персонаж дошёл до края.
Для перехода нужен Суд / Аттестация соответствующего яруса.
```

Переходы команды:

- Физический → Тонкий: Врата Пробуждения.
- Тонкий → Метагалактический: Пороги Духа.
- Метагалактический → Огненный: Испытание Воли.
- Огненный → Синтезный: Суд Огня.
- Синтезный → финал / Экополис: Допуск Экополиса.

Параметры порогов и числовой баланс остаются `TO-PLAYTEST`.
