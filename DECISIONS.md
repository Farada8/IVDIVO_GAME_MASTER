# DECISIONS — журнал решений основателя

Статус: ACTIVE  
Назначение: фиксировать утверждённые, отменённые и изменённые решения проекта.

## Правила

- Только основатель может присвоить статус APPROVED.
- Новая гипотеза не заменяет утверждённое решение без записи в этом файле.
- Каждое фундаментальное изменение получает ID формата `DEC-000`.
- Если решение меняет главный контекст, обновляется `PROJECT_CORE_CONTEXT.md`.

### DEC-009 — Коллективная эволюция стран и планет

Статус: APPROVED IN PRINCIPLE / MECHANICS DRAFT  
Дата фиксации: 2026-07-20  
Источник: Google Drive `03_ЖУРНАЛ_РЕШЕНИЙ_v0.2_ACTIVE`

В игре и вселенной субъектами развития являются не только персонажи, но также страны и планеты. Страна имеет собственный уровень эволюции коллективного Логоса, включая возможные пятый, шестой и седьмой уровни, а её развитие существенно влияет на раскрытие способностей персонажа.

Не утверждает формулу расчёта Логоса, жёсткость ограничений, управление страной и условия планетарного перехода.

### DEC-011 — Порядок построения вселенной: архитектура → философия → базовые практики → отбор → активация

Статус: PROPOSED  
Дата фиксации: 2026-07-11  
Источник: текущий диалог

Работа над вселенной Плерос/LIMINAL проходит через последовательность: архитектура вселенной → философия → базовые практики → отбор из внешних источников → активация в механику.

Элемент, не прошедший этапы 1–4, не может быть активирован в механику напрямую.

### DEC-015 — Пять вертикальных миров + орбитали

Статус: APPROVED  
Дата фиксации: 2026-07-20  
Источник: PROJECT_CORE_CONTEXT_2026 v1.6 / WORLD_ARCHITECTURE_v0.2_APPROVED

Ранняя восьмимировая вертикаль заменена архитектурой пяти вертикальных миров и орбиталей параллельных миров:

1. Физический мир.
2. Тонкий мир.
3. Метагалактический Духо-мир.
4. Огненный мир.
5. Синтезный мир.

Орбитальные параллельные миры: Ад, Рай, Аидия, Демонский, Омарный и другие TO-VERIFY. Орбитали не являются вертикальными ступенями.

Заменяет: DEC-003 как текущую вертикальную архитектуру.

### DEC-016 — Тонкий мир ведёт только в Метагалактический Духо-мир

Статус: APPROVED  
Дата фиксации: 2026-07-20  
Источник: PROJECT_CORE_CONTEXT_2026 v1.6

Тонкий мир по вертикали ведёт только в Метагалактический Духо-мир. Прямой переход из Тонкого мира в Огненный или Синтезный мир отсутствует.

### DEC-017 — Абсолютный Огонь в Метагалактическом Духо-мире

Статус: APPROVED IN PRINCIPLE / MECHANICS DRAFT  
Дата фиксации: 2026-07-20  
Источник: PROJECT_CORE_CONTEXT_2026 v1.6

В Метагалактическом Духо-мире существует клетка хода «Абсолютный Огонь»: игрок стяжает Абсолютный Огонь, выбирает карту-мыслеобраз, направляет Огонь через простое действие и после подтверждения получает возможность перейти в Синтезный мир.

Точные параметры колоды, числа карт, порогов и эффектов остаются DRAFT / HYPOTHESIS.

### DEC-018 — Протокол архивации разработок разговоров

Статус: FOUNDER-DECISION / ACTIVE  
Дата фиксации: 2026-07-20  
Источник: текущий разговор

Каждый содержательно значимый разговор должен быть обработан целиком: разработки извлекаются, очищаются от шума, классифицируются как CANON / WORKING / IDEAS / REJECTED / OPEN QUESTIONS и фактически сохраняются в Google Drive и GitHub проекта. Обновляются тематические документы, индексы, журнал решений и changelog. Нельзя утверждать сохранение без фактической записи и проверки.
