# DECISIONS — журнал решений основателя

Статус: ACTIVE  
Назначение: фиксировать утверждённые, отменённые и изменённые решения проекта.

## Правила

- Только основатель может присвоить статус APPROVED.
- Новая гипотеза не заменяет утверждённое решение без записи в этом файле.
- Каждое фундаментальное изменение получает ID формата `DEC-000`.
- Если решение меняет главный контекст, обновляется `PROJECT_CORE_CONTEXT.md`.

Статус: APPROVED / ACTIVE  
Дата фиксации: 2026-07-07  
Источник: Google Drive, ЖУРНАЛ_РЕШЕНИЙ v0.2 ACTIVE (перенесено в GitHub 2026-07-09)

Перед каждой содержательной задачей необходимо читать `PROJECT_CORE_CONTEXT_2026` и релевантные документы; проверять версии; сохранять значимые результаты; обновлять журнал решений и контекст; после сохранения проверять название, папку, содержимое и ссылку на файл.
