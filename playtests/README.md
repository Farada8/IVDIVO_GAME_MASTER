# playtests

Здесь будут храниться отчёты тестов, метрики, проблемы, изменения баланса и решения по прототипу.

Минимальные поля отчёта:

- дата
- версия прототипа
- количество игроков
- длительность объяснения
- длительность партии
- что было понятно
- что было непонятно
- где возникло напряжение
- где возникла скука
- какие решения были интересными
- что сломало баланс
- что изменить в следующей версии
