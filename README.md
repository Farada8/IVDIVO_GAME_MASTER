# IVDIVO GAME MASTER

Единый репозиторий проекта международной настольной игры и IP-вселенной.

## Текущий статус после cleanup 2026-07-22

Перед любой работой читать:

1. [PROJECT_CORE_CONTEXT.md](PROJECT_CORE_CONTEXT.md)
2. [DECISIONS.md](DECISIONS.md)
3. [DOCUMENT_REGISTRY.md](DOCUMENT_REGISTRY.md)
4. [ROADMAP.md](ROADMAP.md)
5. [OPEN_QUESTIONS.md](OPEN_QUESTIONS.md)

Главные текущие решения:

- Жанр: **Dark Cosmic Adventure**.
- Архитектура: **5 ярусов + орбитальные / параллельные миры**.
- Ресурсы: **Дух / Свет / Огонь**.
- Движение: **кольца / кубик / секторы / аттестации**.
- База игроков: **Экополис IVDIVO**, не старое Убежище.
- Роли: активная рабочая рамка — **16 ролей**.

Старые слои `horror-primary`, `8 worlds`, `Луч как central resource`, `Убежище`, `4 roles` считаются `SUPERSEDED / OLD MODEL`, если не подтверждены заново отдельным DEC.

## Назначение

Этот репозиторий является главным текстовым хранилищем проекта: контекст, решения, исследования, механики, миры, карты, кампании, Kickstarter, бизнес-модель, франшиза, AI-результаты и идеи.

Google Drive остаётся хранилищем для Word/PDF, изображений, презентаций, макетов, прототипов и материалов обмена.

## Главные файлы

- [PROJECT_CORE_CONTEXT.md](PROJECT_CORE_CONTEXT.md) — обязательный текущий контекст проекта.
- [DECISIONS.md](DECISIONS.md) — журнал утверждённых решений основателя.
- [DOCUMENT_REGISTRY.md](DOCUMENT_REGISTRY.md) — реестр активных и устаревших документов.
- [CHANGELOG.md](CHANGELOG.md) — журнал изменений базы знаний.
- [ROADMAP.md](ROADMAP.md) — дорожная карта разработки.
- [OPEN_QUESTIONS.md](OPEN_QUESTIONS.md) — нерешённые вопросы после cleanup.

## Рабочие разделы

- `docs/06_MECHANICS/` — активный рабочий слой механики новой модели.
- `docs/04_CARDS/` — активный слой карточных пакетов и raw CSV.
- `docs/00_PROJECT_CORE/` — governance, протоколы, cleanup-заметки.
- `docs/` — утверждаемые и рабочие документы проекта.
- `ideas/` — новые идеи, гипотезы и наброски до утверждения.
- `inbox/` — входящие заметки, ссылки, сырой материал.
- `templates/` — шаблоны документов.
- `references/` — внешние источники, аналоги, книги, ссылки.
- `playtests/` — тесты, метрики, отчёты балансировки.
- `assets/` — лёгкие вспомогательные материалы; тяжёлые файлы хранить в Google Drive.

## Правило проекта

Новая гипотеза не заменяет утверждённое решение. Все значимые изменения должны фиксироваться в `DECISIONS.md` или `CHANGELOG.md`.

Если документ противоречит DEC-018..022, он считается старым слоем до проверки.
