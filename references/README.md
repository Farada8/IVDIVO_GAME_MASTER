# references — внешние источники и библиотеки

Здесь хранятся индексы источников, ссылки, библиографии, списки аналогов и материалы для исследования.

## Разделы

- board_games_market — рынок настольных игр.
- kickstarter — краудфандинг и кампании.
- source_indices — ИВДИВО, Бейли, Блаватская, Агни-Йога, Роза Мира и другие источники.
- reviews — обзоры, негативные отзывы, Reddit, YouTube.

## Правило

Внешний источник не становится решением проекта сам по себе. Он должен быть переведён в игровую адаптацию или исследовательский вывод.
