## 2. Ядро учения, релевантное проекту

Двадцать два Великих Аркана трактуются не как гадательные карты, а как **последовательные ступени пути духа**: этапы познания и реализации, по которым сознание восходит от природного состояния к синтезу. Опорные конструкции Шмакова: **бинер** (полярная пара), его **нейтрализация в тернер** (рождение синтезирующего третьего), иерархия монад; пневматология как «механика становления духа». Система жёстко связана с каббалистической решёткой (22 ступени ↔ 22 буквы) — прямой стык с запланированным P14.

## 3. Резонансы с утверждённым ядром (подтверждения, не изменения)

Самое сильное совпадение: выбор судьбы Корня. **Разрыв** = уничтожение одного полюса бинера (быстро, со шрамом — полярность не разрешена, а срезана). **Преобразование** = нейтрализация бинера: угроза становится третьим, союзным законом зоны (дороже, но рождает синтез). Уже принятая механика (GCORE-001, P45) оказывается структурно шмаковской без единого заимствованного слова — это внутренняя валидация, менять ничего не требуется. Второй резонанс: сама вертикаль восьми Дисков как «становление духа» — восхождение ценой накопленного (утверждённый космологический тормоз).

## 4. Варианты игровой абстракции «Путь Восхождения» (все HYPOTHESIS, активация только решением DEC-010)

**A — Пороги по всей вертикали:** 22 Порога Восхождения распределены по восьми Дискам (2–3 на Диск) как скрытый каркас кампанейской прогрессии персонажа — работает так же, как юнгианская дуга: структура внутри, имя источника снаружи не звучит.

**B — Трек верхней триады:** Путь активен только в Дисках 6–8 (Душа → Воля → Огненный) — если под «духовным миром» основатель понимает верхнюю триаду; нижние Диски дают подготовку, верхние — собственно восхождение.

**C — Вехи кампании:** 22 ступени = сюжетные вехи полной кампании (стыкуется с P50: 8–16 партий + промежуточные вехи между ними).

**D — Колода Пути для «Древа Отражений»:** 22 карты-ступени как проективный инструмент уровней B/C (PSY-001). Жёсткие условия: гейт арт-терапевта обязателен; инструмент рефлексии, а не гадание и не эзотерический тест (DEC-001, RES-001 §3.9).

Варианты A и C совместимы (Пороги персонажа + вехи кампании — одна решётка, два масштаба); B — уточнение A; D — независимая линия.

## 6. Риски

Массовое восприятие Таро = гадание: буквальное использование бьёт по ритейлу и нарушает DEC-001 — потому только скрытый каркас. Два скрытых скелета прогрессии (юнгианская дуга + шмаковская лестница) могут конфликтовать — нужен один главный: согласование хранитель лора + юнгианский контур, предложение по итогам Блока 2. Для культурного теста 481A слой нейтрален (не кельтский) — не выносить в заявочные материалы.

## 7. Предложение DEC-010 (на утверждение основателя)

(1) Внести Шмакова в реестр источников внутреннего контура (Блок 2, стык с P14). (2) Выбрать вариант абстракции: A / B / C / A+C / D (или комбинация). (3) Утвердить термин для словаря IP: «Путь Восхождения» + «Пороги» (или иной). (4) Подтвердить трактовку «духовного мира»: верхняя триада (Диски 6–8) — предлагается по умолчанию.
