## 1. Founder question

```text
Promotion plan for many board games / tabletop studio, not only one game.
```

## 2. Core promotion thesis

Do not promote only one box.

Promote a small game studio with a clear first hero game and several future game lines:

```text
Studio brand -> visible prototype process -> playtest community -> first hero game -> Kickstarter -> expansions -> smaller spin-off games -> events -> IP ecosystem.
```

The first public message should be simple:

```text
We are building colorful family fantasy games about portals, sparks, treasures, heroes, creatures, and magical adventures.
```

Do not lead public marketing with deep metaphysics. Keep IVDIVO depth as the advanced/lore layer.

## 3. Promotion goals by stage

### Stage 0: visibility before product

Goal:

```text
Make people curious and collect first followers.
```

Time:

```text
Month 1-2
```

Targets:

```text
- 100-300 social followers;
- 20-50 email leads;
- 10-20 local testers;
- 3 playable rough prototypes.
```

Content:

```text
- board sketches;
- AI concept visuals;
- sparks/token tests;
- 30-second turn examples;
- character reveals;
- playtest table photos;
- short videos: "Would you play this?";
- polls between different game concepts.
```

### Stage 1: prototype community

Goal:

```text
Turn followers into testers and early believers.
```

Time:

```text
Month 3-4
```

Targets:

```text
- 300-800 social followers;
- 100-300 email leads;
- 30-50 playtesters/demo players;
- 3-5 good prototype copies.
```

Content:

```text
- playtest clips;
- before/after rule improvements;
- top 5 funny moments;
- portal market tension examples;
- spark economy examples;
- 6 character ability reveals;
- photos of printed prototype;
- community voting for card names/art.
```

### Stage 2: Kickstarter prelaunch

Goal:

```text
Build a warm audience for the first hero game.
```

Time:

```text
Month 5-9
```

Targets:

```text
- 1000-3000 qualified email leads if aiming for USD 30k-50k;
- 5-10 reviewers/previewers;
- 100+ demo players;
- 1 campaign video;
- 1 full playthrough video;
- 1 Kickstarter preview page.
```

Content:

```text
- official trailer;
- how to play in 60 seconds;
- full turn walkthrough;
- component spotlight: Sparks;
- portal mechanic explainer;
- reviewer previews;
- print prototype unboxing;
- countdown posts;
- founder story;
- comparison: why this is not a boring roll-and-move.
```

### Stage 3: campaign launch

Goal:

```text
Convert the warm audience into backers.
```

Campaign focus:

```text
- day 1 momentum;
- clear pledge value;
- visible components;
- family fun;
- short gameplay proof;
- low-risk fulfillment plan;
- stretch goals that do not destroy margin.
```

## 5. Content pillars

The studio needs repeated content categories:

```text
1. Prototype journey — watch the game being born.
2. Table fun — people laughing, choosing, risking, opening portals.
3. Components — Sparks, portals, cards, dice, pawns.
4. Characters — each character has personality and power.
5. Mechanics — simple demos of why the game is not boring.
6. Art evolution — sketch -> AI concept -> refined card -> printed card.
7. Community choices — let audience vote.
8. Founder story — artist in Ireland building a board game studio.
9. Kickstarter prep — transparent progress.
10. Multi-game studio — more games coming from same world/toolkit.
```

## 6. Weekly content rhythm

Minimum sustainable rhythm:

```text
3 short videos per week.
2 image/carousel posts per week.
1 dev diary/update per week.
1 community question/poll per week.
1 playtest/demo action every 2 weeks.
```

Example week:

```text
Monday: show one card or character.
Tuesday: 30-second rule/mechanic demo.
Wednesday: poll: choose portal/card art/name.
Thursday: prototype progress photo.
Friday: short playtest moment.
Weekend: local test/demo or longer dev update.
```

## 7. Lead magnet

To collect emails, offer:

```text
- free print-and-play mini-game: Sparks;
- downloadable 6-card preview pack;
- prototype rules PDF;
- coloring page / family activity page;
- early access to playtest.
```

Best first lead magnet:

```text
Sparks: a 15-minute mini card game using the same Spark tokens and portal world.
```

## 8. Promotion of multiple games

Do not confuse audience by promoting 10 games equally.

Public priority:

```text
Hero Game: Portals.
Side Mini-Game: Sparks.
Future Teasers: Little Dragons, Portal Market, IVDIVO Orbitals.
```

Structure:

```text
70% content = Portals.
20% content = Sparks mini-game / companion products.
10% content = future studio ideas.
```

## 9. Local launch path in Ireland

Actions:

```text
- make printed prototype;
- run small playtests with families/friends;
- contact local board game cafes/groups;
- contact game shops for feedback;
- take photos and testimonials;
- attend local markets/conventions if accessible;
- build credibility as Ireland-based creator.
```

Possible demo positioning:

```text
Family fantasy board game prototype looking for local playtesters.
```

## 10. Reviewer / previewer strategy

Only contact reviewers after the game is playable.

Prepare:

```text
- one-page sell sheet;
- rules PDF;
- 5-minute overview video;
- prototype photos;
- component list;
- clear target audience;
- honest prototype status.
```

Start with small/mid reviewers, not only top channels.

## 11. Paid ads

Do not start heavy ads before the page and prototype are ready.

Testing plan:

```text
Stage 1: EUR 5-10/day tests on visuals.
Stage 2: promote lead magnet / email signup.
Stage 3: promote Kickstarter prelaunch page.
```

Ad should show:

```text
- board on table;
- colorful Sparks;
- portal mechanic;
- family/adventure promise;
- clear call-to-action.
```

## 12. Conversion funnel

Recommended funnel:

```text
Short video / post -> landing page -> email signup -> prototype updates -> Kickstarter prelaunch follow -> launch day pledge.
```

Do not rely on social followers alone. Email/prelaunch followers matter more.

## 13. Promotion KPIs

Track weekly:

```text
- social followers;
- email leads;
- playtesters;
- comments/questions;
- landing page conversion;
- cost per email lead;
- Kickstarter prelaunch followers;
- reviewer contacts;
- demo event signups.
```

Minimum green-light before Kickstarter:

```text
- 500+ email/prelaunch followers for small launch;
- 1500-3000 for stronger USD 30k-50k target;
- 100+ demo/playtest players;
- 5+ preview/reviewer contacts.
```

## 14. First 30 days promotion plan

```text
Week 1:
- announce studio/game journey;
- post 3 board visuals;
- create simple landing page;
- open email signup;
- ask people to choose favorite board style.

Week 2:
- show Sparks tokens;
- show 3 character ideas;
- post one 30-second turn example;
- recruit first 5-10 testers.

Week 3:
- show printed paper prototype;
- run first test;
- post what was fun/boring;
- adjust rules publicly.

Week 4:
- reveal mini-game Sparks;
- offer PNP signup;
- collect emails;
- choose next prototype to develop.
```

## 15. Main rule

Do not market the project as abstract ideas.

Market visible proof:

```text
board on table;
people playing;
Sparks in hand;
portal cards in market;
player choices;
funny risk moments;
before/after iteration.
```
