## 1. Purpose

This prompt bank supports development of a multi-game tabletop studio, not only one board game.

Use prompts for:

```text
- Portals hero game;
- Sparks mini-card game;
- Portal Market card game;
- Little Dragons / Companions;
- Treasure Rings;
- Shipwreck Island;
- Fairy Tricks;
- Monster Helpers;
- IVDIVO Orbitals advanced game;
- Great Portal Campaign / co-op system.
```

## 2. Prompt 1 — portfolio ideation

```text
Act as a professional tabletop publishing strategist. Create 20 board/card game concepts for a small family-fantasy tabletop studio. Each concept must have: title, target audience, player count, playtime, core hook, key components, production cost level, Kickstarter potential, retail potential, risk, and first prototype method. Prioritize games that can become a product line, share art/IP/components, and be tested cheaply.
```

## 3. Prompt 2 — select the best 3 games

```text
Act as a tabletop studio greenlight committee. Evaluate the following game concepts and select the best 3 to prototype first. Use criteria: fun in 15 minutes, production cost, visual appeal, family clarity, replayability, Kickstarter hook, expansion potential, ease of playtesting, and compatibility with Portals/Sparks/IP universe. Give scores from 1-10 and a clear recommendation.
```

## 4. Prompt 3 — one-page game concept

```text
Create a one-page concept sheet for the board/card game [GAME TITLE]. Include: elevator pitch, player fantasy, target audience, player count, playtime, components, turn structure, core loop, victory condition, emotional hook, table presence, why it is not boring, first 48-hour paper prototype plan, and what to test first.
```

## 5. Prompt 4 — 48-hour paper prototype

```text
Design a 48-hour paper prototype for [GAME TITLE]. Keep it cheap and fast. Define exact components to print or handwrite: board, cards, tokens, dice, player aids. Include card counts, token counts, basic rules, first test scenario, and 10 questions to answer after the first playtest.
```

## 6. Prompt 5 — Portals hero game rules

```text
Act as a professional family board game designer. Develop v0.1 quick rules for Portals: a colorful family fantasy game with a 3-ring board, magical Sparks, portal cards, hero cards, event cards, treasure cards, creatures, two dice, and 2-6 players. Keep rules playable in 30-45 minutes, age 8+, with clear choices each turn: move, risk, collect Sparks, play cards, open portals, race to the Great Portal. Output a 2-page quick rules draft.
```

## 7. Prompt 6 — Portals board cell distribution

```text
Create a balanced 3-ring board cell distribution for Portals. Outer ring = travel/resources/safe adventure. Middle ring = portals/trials/danger/creatures. Inner ring = high reward/victory risk. Give exact number of cells per type, example cell names, icon type, card deck triggered, reward/risk, and how each ring creates escalation. Avoid boring roll-and-move spaces.
```

## 8. Prompt 7 — Sparks economy

```text
Design the Spark economy for a family fantasy board/card game. There are 6 Spark colors: Gold, Blue, Green, Red, Purple, White. Define what each color does, how players gain them, how they spend them, conversion rules, scarcity, victory use, card costs, portal costs, and anti-runaway balancing. Make the economy simple for children but interesting for adults.
```

## 9. Prompt 8 — character cards

```text
Create 12 character cards for Portals. Each character must have: name, family-friendly fantasy archetype, starting Sparks, movement ability, portal ability, treasure/risk ability, once-per-game power, personality hook, weakness, and simple icon language. Keep them asymmetric but balanced for age 8+.
```

## 10. Prompt 9 — portal card deck

```text
Create 40 Portal cards for Portals. Each card must include: title, color/type, Spark cost, light points, immediate effect, persistent bonus or discount, family-friendly flavor, balance note, and whether it is common/uncommon/rare. The cards should create an open market row of 5 cards with Splendor-like tension: buy now or wait and risk another player taking it.
```

## 11. Prompt 10 — event card deck

```text
Create 40 Event cards for Portals. Events should create fun table drama without being cruel. Include festivals, storms, moving portals, bridges opening/closing, creatures moving, market changes, shared rewards, light chaos, and catch-up effects. Each card needs: title, trigger, effect, affected players, duration, family safety note, and anti-frustration rule.
```

## 12. Prompt 11 — treasure/adventure deck

```text
Create 40 Treasure and Adventure cards for a family fantasy board game. Half should be rewards/tools, half should be mini-challenges. Each Adventure card should offer a safe option and a risky option. Include: title, illustration prompt, safe choice, risky choice, success reward, failure consequence, Spark interaction, and recommended age clarity.
```

## 13. Prompt 12 — creatures and allies

```text
Create 30 Creature/Ally cards for Portals. Each creature can be fought, befriended, traded with, or avoided. Include friendly creatures, mischievous creatures, rival monsters, and helpers. Each card needs: name, type, mood, test value, reward, befriend cost, danger result, ally power, and family-friendly flavor.
```

## 14. Prompt 13 — custom dice

```text
Design a simple custom dice system for Portals. There is a movement die and a risk die. Define all faces, probabilities, when to roll each die, cards that modify dice, reroll rules, push-your-luck options, and how dice create excitement without making the game random and unfair.
```

## 15. Prompt 14 — small card game Sparks

```text
Design a 15-minute small card game called Sparks, using the same 6 Spark colors from Portals. It should be cheap to print, easy to teach, 2-5 players, age 7+, with set collection, light stealing/trading, and combo cards. Output components, setup, turn structure, 60-card deck list, victory condition, and why it can be a lead magnet for Kickstarter.
```

## 16. Prompt 15 — Portal Market game

```text

It should be similar in tension to engine-building/set-collection games but original in theme. Output full rules, card types, deck structure, 30 sample cards, and balancing principles.
```
