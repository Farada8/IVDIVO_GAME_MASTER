# DEC-016 — Тонкий мир ведёт только в Метагалактический Духо-мир

Дата: 2026-07-12  
Статус: APPROVED / FOUNDER-DECISION  
Источник: прямое решение основателя в текущем диалоге  
Связанные документы: WORLD-002, MECH-003, MECH-004

## Утверждённый закон

> **По вертикальной оси восхождения Тонкий мир имеет только один выход: в Метагалактический Духо-мир.**

Из Тонкого мира не существует прямого вертикального перехода:

- в Огненный мир;
- в Синтезный мир;
- через Семинар ИВДИВО непосредственно в Синтезный мир;
- через ИВО Огонь Поядающий;
- через Аркан «Суд».

Последние три пути могут вести в Синтезный мир только из тех миров или состояний, которые будут отдельно утверждены. Они не являются выходами из Тонкого мира.

## Вертикальная последовательность

```text
Физический мир
      ↓
Тонкий мир
      ↓  единственный вертикальный выход
Метагалактический Духо-мир
```

Дальнейшие пути после Метагалактического Духо-мира проектируются отдельно. Уже зафиксированные рабочие направления:

- путь через Метагалактический Духо-мир и Аркан «Суд»;
- путь из Метагалактического Духо-мира в Огненный мир, далее через ИВО Огонь Поядающий;
- путь через Семинар ИВДИВО, точное исходное положение и условия которого остаются TO-VERIFY.

## Функциональное различие

Тонкий мир:

- раскрывает образы, состояния, связи и Искажения;
- показывает, как духовный закон переживается и искажается;
- не назначает окончательную роль физике;
- не даёт прямого доступа к Синтезному миру.

Метагалактический Духо-мир:

- является следующим миром после Тонкого;
- содержит духовные законы и назначения ролей;
- воздействует на физику через Арканы Таро;
- является обязательным узлом перед духовными путями к Огненному и Синтезному мирам.

## Последствия для игры

1. Любая карта или сценарий прямого перехода `Тонкий → Огненный` или `Тонкий → Синтезный` получает статус REJECTED / SUPERSEDED.
2. Тонкий мир не должен содержать финальные карты Суда или Синтеза как собственные выходы.
3. Завершение Тонкого мира должно открывать вход только в Метагалактический Духо-мир.
4. Орбитальные переходы рассматриваются отдельно и не меняют этот закон вертикали.
5. WORLD-002 требуется уточнить этим решением при следующей версии архитектуры.
