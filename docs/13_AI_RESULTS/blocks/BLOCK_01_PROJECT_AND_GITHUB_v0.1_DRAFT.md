## 1. Аудит репозитория

Главные найденные коллизии:

- ранний DEC-003 и WORLD-001 описывают восемь вертикальных миров;
- более поздние решения используют пять вертикальных уровней и отдельные параллельные миры;
- CARD-001 и часть NARR-002 всё ещё опираются на старую модель;
- прогрессия MECH-004/CARD-002 новее реестра и требует регистрации;
- несколько GitHub PR образуют stacked-цепочку и не должны сливаться в случайном порядке;
- Google Drive содержит архивные дубликаты протоколов и журналов решений.

Решение: старые документы не удалять; помечать точечные положения `SUPERSEDED` после нового решения основателя.

## 4. Источник истины

Порядок:

1. прямое решение основателя;
2. более поздний FOUNDER-DECISION;
3. APPROVED;
4. REVIEW;
5. DRAFT;
6. AI PROPOSAL.

Числа не получают APPROVED без теста. Канонические имена не создаются ИИ без источника.

## 5. Зависимости

Критический граф:

`CANON → BASE SCOPE → CORE LOOP → ROLE/CARD PROTOTYPE → PLAYTEST → CAMPAIGN → EXPANSIONS → ART/PRODUCTION → KICKSTARTER`.

Нельзя проектировать 50 финальных иллюстраций до блокировки функций карт. Нельзя балансировать дополнение до работающей базы.

## 6. Журнал решений

Нужна новая запись со статусом `PROPOSED / FOUNDER DIRECTION`:

> Базовая коробка является законченной игрой; полные вертикальные уровни и параллельные миры вводятся модульными дополнениями после освоения ядра.

Это направление не должно автоматически объявляться финальной космологией.

## 7. Гейты

- Gate A: канон не содержит активных противоречий.
- Gate B: два игрока проходят обучающий сценарий без помощи автора.
- Gate C: core loop даёт содержательное решение каждые 3–5 минут.
- Gate D: портал создаёт выбор без вынужденного ожидания.
- Gate E: первое открытие меняет старую карту и используется в следующем сценарии.
- Gate F: минимум 40 слепых партий перед Kickstarter-разработкой финального масштаба.

Создать трёхслойный словарь:

Запрещено переносить эзотерическую терминологию напрямую в массовый продукт без решения.

Обязательны шаблоны: механика, мир, дополнение, персонаж, фракция, карта, сценарий, тест, art brief, production spec.
