## ТРИ ОСИ АРХИТЕКТУРЫ (полная картина)

Теперь структура мира имеет ТРИ независимых измерения из источника:

| Ось | Шкала | Вопрос | Документ |
|---|---|---|---|
| **РОЛЬ** | 19-рица (Человек…Отец) | КТО действует | ROLES_ARCHITECTURE, STRUCTURE_32 |
| **МАСШТАБ** | 15-рица (Метагалактический…Си-ИВДИВО) | ГДЕ (уровень мироздания) | SCALE_15_ARCHETYPES |
| **МАТЕРИЯ** | 64 вида (Физическая…Сиаматическая) | ИЗ ЧЕГО (субстанция) | этот документ |

**Позиция сущего = [Роль] действует на [Масштабе] с [Видом материи].**
Пример: Теург (роль) на Метагалактическом (масштаб) оперирует Астральной материей (вид) — конкретное определённое действие.

Это объясняет источниковое «управление 64 видами материи»: не 64 предмета, а 64 РЕГИСТРА субстанции, с которыми работает роль в зависимости от масштаба и охвата.

## СВЯЗЬ С ПРИНЯТЫМ В ПРОЕКТЕ
- Зона A (1–7) = основа синтез-миров: Физическая→Астральная→Причинная соответствуют Тверди/Спектру/Выси (WORLD_ADAPTATION_MATRIX). Классический слой, которым и надо начинать.
- «64 частности» / «64 частей» из регламентов (Р11, Р33) = отражение 64 видов материи в человеке (микрокосм). Антропный принцип: 64 макро → 64 микро.
- Симметрия огня (низ жжёт / верх преображает) работает вдоль этой шкалы: Физическая (низ) ↔ высшие виды.

## МЕХАНИЧЕСКИЙ ПОТЕНЦИАЛ И ПРЕДЕЛ
1. **64 вида целиком неигрибельны** на 40–60 картах — как 15 масштабов и 66 центров. Полная шкала = канон/лор. Играбельный срез — Зона A (7 классических планов) или её часть (2–3 плана старта, ранее принято).
2. **Зоны C (17–64)** — авторская терминология без аналогов; для игры либо абстрагировать (цвета/типы), либо оставить как поздний контент/лор глубины.
3. **Материя как ресурс-тип:** каждый вид = регистр, в котором роль может действовать. Аналог «стихий» Spirit Island, но 64 вместо 4-6 → нужен свёрнутый играбельный набор.
4. **Кодоны генетики (64):** источник связывает 64 материи с 64 кодонами ДНК. Потенциальная эстетика/нарратив (материя как «генетический код мира»), но НЕ научный факт — подавать как игровую мифологию.

## GOVERNANCE
- DRAFT. Прямая цитата источника. Названия видов (Зона C) — терминология ИВДИВО, под заменой при переносе в игровой слой; Зона A (Физическая…Атмическая) — общеэзотерические термины, менее проблемны.
- Три оси (роль/масштаб/материя) — каркас канона. Играбельный срез каждой оси — решение Yar.
- «64 = кодоны ДНК» — мифология, не наука. Не подавать как факт.
