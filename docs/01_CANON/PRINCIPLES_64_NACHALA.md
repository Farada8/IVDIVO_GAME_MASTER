# 64 НАЧАЛА
### Ось качества действия — шестое измерение канона

> **Статус:** DRAFT · Block 1 (Canon) · прямая цитата источника (Yar)
> **Назначение:** список 64 качеств/принципов, которыми совершается действие. Отличается от предыдущих пяти шкал по РОДУ: не уровень (кто/где/из чего/какое царство/какой виток), а КАКИМ КАЧЕСТВОМ.

## РОД ЭТОЙ ОСИ — ОТЛИЧИЕ ОТ ПРЕДЫДУЩИХ ПЯТИ

| Ось | Вопрос |
|---|---|
| Роль (19-рица) | КТО действует |
| Масштаб (15-рица) | ГДЕ (уровень мироздания) |
| Материя (64 вида) | ИЗ ЧЕГО (субстанция) |
| Космос (8 царств) | КАКОЕ ЦАРСТВО |
| Эволюция (64=16×4) | КАКОЙ ВИТОК |
| **Начала (64)** | **КАКИМ КАЧЕСТВОМ** |

Начала — не вертикаль уровней, а ПАЛИТРА КАЧЕСТВ. Любая роль на любом масштабе может действовать через разные Начала: Теург правит стихиями Началом Мощи или Началом Любви — разное по характеру действие при том же охвате.

## СВЯЗЬ С МЕХАНИКОЙ

Зона D (49–64: Мысли, Чувства, Ощущения, Движения) — самые «личные», конкретные качества. Это может быть НИЖНИЙ, стартовый регистр действия игрока (то, чем управляет любой новичок), тогда как Зона A (Синтез, Воля, Творение) — регистр высших ролей.

**Потенциальная механика:** Начало = МОДИФИКАТОР действия, а не отдельный ресурс. Одно и то же действие роли (пробудить, воскресить, разомкнуть) окрашивается конкретным Началом — как «через что» оно совершено. 64 Начала целиком неигрибельны; играбельный срез — 6–8 ключевых (по одному представителю зоны, либо привязка к 8 космосам/8 ролям).

**Гипотеза стыковки (TO-VERIFY):** 64 Начала ↔ 64 материи ↔ 64 эволюции — три параллельных списка по 64. Возможно, это ТРИ ГРАНИ ОДНОЙ ШКАЛЫ (эволюция = виток, материя = из чего сделан этот виток, начало = каким качеством он движим). Требует сверки позиция-к-позиции, если понадобится точная стыковка.

## GOVERNANCE
- DRAFT. Прямая цитата источника. Термины (Ивдивность, Синтезначала, Имперации, Окскость) — специфическая терминология ИВДИВО, под заменой при переносе в игру. Общие (Мысли, Чувства, Любовь, Форма, Время) — нейтральны.
- Полный список — канон/лор. Играбельный срез — малый набор качеств-модификаторов, решение Yar.
