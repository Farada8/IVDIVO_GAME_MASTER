## Статус
Все документы **DRAFT / IDEA**. Ни один не APPROVED — статус ставит только Yar.

## Ключевые принятые решения
1. Все роли управляют **миром, не игроками** (защита от runaway на уровне архитектуры).
2. Обе вершины обезврежены: Иерарх держит (не действует), Творец собирает (не бьёт).
3. Единый ресурс **Луч**, меняющий знак по вертикали (Извержение внизу / Исток вверху).
4. **Кубиков нет** — неопределённость из колоды.
5. Прогрессия — **дискретные Пороги**, не XP.
6. Ни одна карта Выси не имеет эффекта без второй строки (размен обязателен).
7. Порча поднимается только тем путём, что открыли сверху (культы).

2. Одновременность миров: настоящая или переключаемая?
3. Сколько слоёв в базовой коробке: 2, 3 или 5?
4. Жители Глуби — предложены Claude, **не заданы автором**. Требуют подтверждения.
