## 3. World architecture

Current approved architecture uses:

```text
5 vertical worlds + orbital/parallel worlds.
```

Vertical worlds:

```text
1. Physical World
2. Thin World
3. Metagalactic / Spiritual-Metagalactic World
4. Fire World / Fire Transition
5. Synthesis World
```

Current orbital / parallel world schema:

```text
Physical world orbitals:
- Paradise
- Fey / leprechaun world
- Unified lower world: Hell + Demonic + Omaric

Thin world orbitals:
- World of Ray Teachers and Hierarchies
- World of Ashtar and alien civilizations / Sirius
- World of Angelic civilization and the Lord of Heaven

Metagalactic-Spirit orbitals:
- Archetypes / Arcana, 22 positions
- Mahatmas / 64 Spirits before the Throne / Sanat Kumaras
- Elohim / Sephiroth
- Allah / 99 Names
- Christ / Maitreya
```

Important rule:

```text
Cells/sectors inside a world are not the same as orbitals.
Orbitals are parallel rings/worlds that enter into relations with the main world circles.
```

## 4. Numerical board structure

Current working direction:

```text
Main disks: 64 positions/cells.
Orbitals: variable size.
16 = baseline/test orbital size.
22 = special Arcana orbital size.
Do not say all orbitals are fixed at 16.
```
