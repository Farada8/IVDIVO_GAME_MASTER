# GAME_01_PORTALS_OF_STARS

Status: ACTIVE WORKING FOLDER / DRAFT  
Project: IVDIVO — Game 01 «Порталы Звёзд»

## Purpose

This folder contains the first small commercial game inside the IVDIVO universe: a family-accessible, competitive portal-building board/card game for 2–5 players, 30–40 minutes, target weight 1.8–2.0.

This is not the future large IVDIVO Core Box. It is the first fast, testable, sellable entry product.

## Current fixed direction

1. Working title: **IVDIVO: Порталы Звёзд / Portals of Stars**.
2. Core loop: collect Sparks, move on the map, activate a location, open/use a portal, complete secret routes, score Light.
3. Theme: portals do not create new power; they help places remember an already existing wholeness.
4. Economy: 4 basic Spark types + 1 joker.
5. Main competition: portal network, secret routes, soft income from other players using your portals.
6. No evil PvP and no mechanics that punish the trailing player.
