## 1. Назначение

Этот документ фиксирует чистую текущую схему вертикальных миров и орбитальных / параллельных миров проекта IVDIVO Card Game.

Документ нужен как рабочая карта для следующих разговоров о:

- мирах;
- орбиталях;
- клетках / секторах;
- персонажах;
- картах;
- переходах;
- конъюнкциях;
- событиях;
- развитии персонажа и кампании.

## 2. Главное разделение

```text
ОСНОВНОЙ МИР = вертикальный уровень.
КЛЕТКИ / СЕКТОРА = внутренние состояния или зоны на круге мира.
ОРБИТАЛИ = параллельные миры, которые входят кругами в основной мир и друг в друга.
```

Это разделение обязательно для дальнейшей разработки: нельзя смешивать клетки мира и орбитальные параллельные миры.

## 3. Общая вертикальная схема миров

```text
5. СИНТЕЗНЫЙ МИР
4. ОГНЕННЫЙ МИР / ОГНЕННЫЙ ПЕРЕХОД
3. МИР МЕТАГАЛАКТИЧЕСКОГО ДУХА
2. ТОНКИЙ МИР
1. ФИЗИЧЕСКИЙ МИР
```

## 4. Полная текущая схема миров и орбиталей

```text
СИНТЕЗНЫЙ МИР
└─ орбитали пока не названы / не утверждены

ОГНЕННЫЙ МИР / ОГНЕННЫЙ ПЕРЕХОД

├─ Орбиталь Архетипов / Арканов
│  └─ отдельное уточнение: 22 позиции / хода
├─ Орбиталь Махатм / 64 Духов пред Престолом / Санат Кумаров
├─ Орбиталь Элохимов / Сефир

├─ Мир Учителей Лучей и их Иерархии

├─ Рай
├─ Мир фей и лепреконов
└─ Объединённый нижний мир: Ад + Демонский + Омарный
```

## 5. Физический мир

Физический мир имеет три текущие орбитали:

1. **Рай**
2. **Мир фей и лепреконов**
3. **Объединённый нижний мир: Ад + Демонский + Омарный**

```text
ФИЗИЧЕСКИЙ МИР
├─ Рай
├─ Мир фей и лепреконов
└─ Объединённый нижний мир
   ├─ Ад
   ├─ Демонский мир
   └─ Омарный мир
```

### Уточнение

Ад, Демонский мир и Омарный мир сейчас не считаются тремя отдельными физическими орбиталями. Они объединены в один нижний комплекс / нижний мир.

Аидия заменена на Мир фей и лепреконов.

## 6. Тонкий мир

Тонкий мир имеет три текущие орбитали:

1. **Мир Учителей Лучей и их Иерархии**
2. **Мир Аштара и инопланетных цивилизаций / Сириус**
3. **Мир цивилизации Ангелов и Господа Небесного**

```text
ТОНКИЙ МИР
├─ Мир Учителей Лучей и их Иерархии
├─ Мир Аштара и инопланетных цивилизаций / Сириус
└─ Мир цивилизации Ангелов и Господа Небесного
```

### Уточнение

Образы, желания, тонкие формы, световое знание, память, сон, связь, искажение и похожие элементы могут быть клетками / секторами на круге Тонкого мира, но не являются орбитальными параллельными мирами Тонкого мира, если основатель не решит иначе.

## 7. Мир Метагалактического Духа

Мир Метагалактического Духа имеет пять текущих орбиталей:

1. **Орбиталь Архетипов / Арканов**
2. **Орбиталь Махатм / 64 Духов пред Престолом / Санат Кумаров**
3. **Орбиталь Элохимов / Сефир**
4. **Орбиталь Аллаха / 99 Имён**
5. **Орбиталь Христа / Майтреи**

```text
МИР МЕТАГАЛАКТИЧЕСКОГО ДУХА
├─ Орбиталь Архетипов / Арканов
├─ Орбиталь Махатм / 64 Духов пред Престолом / Санат Кумаров
├─ Орбиталь Элохимов / Сефир
├─ Орбиталь Аллаха / 99 Имён
└─ Орбиталь Христа / Майтреи
```

### Уточнение по Орбитали Арканов

Орбиталь Архетипов / Арканов имеет отдельное рабочее числовое уточнение:

```text
Орбиталь Архетипов / Арканов = 22 позиции / хода.
```

Полный список 22 Арканов находится в отдельном документе:

```text
docs/05_WORLDS/ARCANA_ORBITAL_SHMAKOV_22_v0.1_SOURCE_DRAFT.md
```

## 8. Синтезный мир

```text
СИНТЕЗНЫЙ МИР
└─ орбитали пока не названы / не утверждены
```

Синтезный мир сейчас важен как верхний уровень, из которого возможны действия сверху вниз:

- столп;
- развёртывание удачи;
- перепись закона;
- сотворение новой формы;
- нисходящий канал;
- временный вход через карту `Рождение Свыше`.

Но отдельные орбитальные / параллельные миры Синтезного мира пока не названы.

## 9. Огненный мир / Огненный переход

```text
ОГНЕННЫЙ МИР / ОГНЕННЫЙ ПЕРЕХОД
└─ орбитали пока не названы / не утверждены
```

Огненный мир сейчас работает как уровень прожига, переплавки и перехода через невозможное. Отдельные орбитали Огненного мира пока не названы.

## 10. Что пока НЕ входит в текущую зафиксированную схему орбиталей

Пока не входят как утверждённые орбитали:

```text
- Орбиталь супергероев
- Орбиталь Асгарда / северных богов
- Орбиталь драконов
- Орбиталь животных-помощников
- Орбитали Огненного мира
- Орбитали Синтезного мира
```

Эти элементы могут быть кандидатами, дополнениями или будущими коммерческими слоями, но в текущей схеме они не являются утверждёнными орбитальными параллельными мирами.

## 11. Рабочая числовая структура

Основные диски миров сейчас рассматриваются как 64-клеточные:

```text
Основной диск мира = 64 клетки / хода.
```

Орбитали имеют переменный размер:

```text
16 клеток = стандартная / тестовая размерность многих орбиталей.
22 клетки = Орбиталь Архетипов / Арканов.
N = допустимая переменная размерность для других орбиталей после уточнения основателем.
```

Важно: правило `16` не должно автоматически применяться ко всем орбиталям.

## 13. Правило использования в проекте

В следующих разговорах и документах нужно использовать эту схему как базовую карту миров и параллельных миров.

Нельзя:

- начинать карту миров заново без сверки с этим документом;
- смешивать клетки мира и орбитали;
- автоматически добавлять супергероев, Асгард, драконов или помощников как утверждённые орбитали;
- превращать draft-эффекты в canon без решения основателя;
- считать все орбитали 16-клеточными.

Можно:

- разрабатывать клетки основных дисков;
- разрабатывать карты орбиталей;
- уточнять размеры орбиталей;
- вводить новые орбитали только после отдельного founder-input;
- создавать игровые эффекты как DRAFT.
