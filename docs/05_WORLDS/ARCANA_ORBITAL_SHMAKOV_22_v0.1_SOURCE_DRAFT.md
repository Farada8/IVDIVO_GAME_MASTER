## 1. Назначение

Зафиксировать рабочий список 22 позиций для Орбитали Архетипов / Арканов, входящей в Мир Метагалактического Духа.

Важно: этот документ НЕ утверждает игровые эффекты Арканов и НЕ заменяет канон. Он нужен как первичная разметка орбитали на 22 позиции, которую затем можно разворачивать в карты, клетки, события, ресурсы, законы и конъюнкции.

## 2. Важная оговорка по Шмакову

У Шмакова структура последних Арканов имеет особую нумерационную логику: источники фиксируют связку `Аркан 0 / XXI / XXII`, а также различение Аркана XXI и Аркана XXII как верховного бинера / тернера.

Поэтому список ниже является рабочим `SOURCE-DRAFT`: традиционные названия используются как навигационные метки, а точные формулировки Шмакова, философские доктрины и игровое назначение каждого Аркана требуют отдельной углублённой сверки.

## 4. Рабочий принцип для игры

Орбиталь Арканов не должна работать как частая малая орбиталь на 16 позиций. Это высокая орбиталь Метагалактического Духа.

Предлагаемый режим:

```text
Орбиталь Арканов = 22 позиции.
Основной диск мира = 64 позиции.
Совпадения 64/22 редкие и значимые.
Аркановое событие меняет не только ресурс, а роль, закон, судьбу, миссию или направление кампании.
```

## 5. Типы карт на основе Арканов — DRAFT

Каждый Аркан может дать минимум 4 карты:

```text
1. Карта Аркана — основная позиция орбитали.
2. Карта Испытания Аркана — цена / риск / искажение.
3. Карта Дара Аркана — положительное раскрытие.
4. Карта Закона Аркана — кампанийное изменение правила.
```

Для 22 Арканов это даёт:

```text
22 позиции орбитали.
88 базовых карт Аркановой линии.
```

## 6. Что НЕ делать без отдельного решения

- Не превращать Арканы в гадание.
- Не копировать чужие иллюстрации Таро.
- Не смешивать Шмакова, Rider-Waite, Марсельское Таро и Кроули без маркировки источников.
- Не утверждать игровые эффекты как APPROVED.
- Не использовать религиозные / оккультные формулировки в публичном массовом слое без отдельного коммерческого переименования.
