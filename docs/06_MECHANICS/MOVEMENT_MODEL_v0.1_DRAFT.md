### Как движется игра: кольца-волвелле, кубик, зоны-секторы

> **Назначение:** зафиксировать модель движения и природу «Зоны», выработанную совместно. Заменяет неопределённость термина «Зона».

## РЕШЕНИЕ: ЧТО ТАКОЕ ЗОНА
Зона = **сектор кольца** (не абстрактное «место», не только карта). Двойственна:
- часть секторов **напечатана на кольцах** (влезшие на поле);
- остальные — **в колоде карт-секторов**, входят на поле ПО ПРАВИЛУ (замена отработанного), не случайным вбросом.
Поле живёт и меняется, но предсказуемо.

## ПОЛЕ
Концентрические вращающиеся кольца (волвелле / Tzolk'in-style). Каждое кольцо = мир:
Физический (центр-ось) · Тонкий · Метагалактический · Огненный · Синтезный.
Игрок — в ЦЕНТРЕ, неподвижная ось (Логика 1). Кольца вращаются вокруг него.
Нарративное обоснование: «игрок — неподвижный центр, реальность вращается; конъюнкция = вспоминание себя на ярусе». Движение = вращение мира вокруг игрока, НЕ беготня игрока.

## ТРИ ДВИГАТЕЛЯ (по одной функции каждый)

### 1. Кубик — ЕДИНСТВЕННАЯ случайность
Бросок → ход/действия игрока на раунд. Источник напряжения. Только он несёт рандом.

### 2. Кольца — ПРЕДСКАЗУЕМАЯ головоломка (НЕ рулетка)
Вращаются по фиксированным взаимно-простым периодам (2,3,5,7 — канон). Игрок ЗНАЕТ, когда кольца сойдутся в окне конъюнкции → планирует наперёд. Тайминг-головоломка = интеллектуальный слой, противовес кубику.
Вращение инициируется/тактируется определёнными картами, НО по предсказуемым периодам (карта = «тик», не случайный бросок).

### 3. Накал — управляемый подталкиватель
Игрок платит Накал → доворачивает кольцо на 1-2 сектора раньше. Дозированный контроль (ресурс ограничен), БЕЗ второго движения (нет двойной арифметики Л2). Механика роли «Смотритель Колец».

## ИСТОЧНИКИ СЛУЧАЙНОСТИ: РОВНО ОДИН (кубик)
Кольца предсказуемы, замена секторов — по правилу. Обоснование: Costikyan «Uncertainty in Games» — избыток источников случайности убивает ощущение своих решений. Один рандом + мастерство планирования колец = «uncertainty as skill».

## ЗОНА-СЕКТОР: ПОЛЯ КАРТЫ (из запроса Yar, к доработке)
Стабильная сторона / искажённая сторона (оборот) / действие зоны / риск / ресурс (по RESOURCE_MODEL: Физ→энергия, Тонкий→Свет, Метагал→Дух, Огненный→Огонь~) / связь с порталом (вскрытие в окне конъюнкции) / связь с Корнем (скрытый жетон ⚠ TBD).

## УРОВНИ СЕКТОРОВ — ЭВОЛЮЦИЯ ПОЛЯ (Yar, 2026-07-22)

Сектор имеет УРОВНИ (базовый → 2…10). Поле не статично — растёт вверх по мере заслуг игроков.

**Триггер подъёма — КОЛЛЕКТИВНАЯ ЗАСЛУГА (не рандом, не таймер):** когда игроки СВОИМИ ЗАСЛУГАМИ переводят мир (или орбиталь) на следующий уровень, открывается КОЛОДА КАРТ этого уровня (2–10) для данного мира/орбитали. Новые сектора-карты **выкладываются ПОВЕРХ старых прямо на поле** (накладка — видно, что мир «поднялся»).

**Три независимых измерения одного сектора (сложившаяся глубина поля):**
| Слой | Что меняет | Триггер |
|---|---|---|
| Сторона (стаб./искаж.) | здоров/заражён | Порча/Разлом |
| **Уровень (1→10)** | **мир поднялся на ступень** | **заслуги игроков (командно)** |
| Вращение | какой сектор напротив | периоды колец |

**Ludonarrative:** механика = тема. Игра про восхождение/преобразование миров → поле буквально растёт по уровням. «Преобразование зоны» как цель = подъём уровня поля (ответ на вопрос, висевший в FORMS_50).

**РИСК ОБЪЁМА (не разрешён):** колоды 2–10 × ~6 сущностей × ~7 секторов ≈ до ~380 карт секторов — неподъёмно для Core Box. МАСШТАБ НЕ ВЫБРАН (Yar: «запиши, масштаб потом»). Кандидаты решения:
- A: Core Box уровни 1–3 картами, выше — допы.
- B: уровни жетонами-накладками (усиливают существующий сектор, не новые карты).
- C: уровни только у якорных секторов (портал/особое).
Рекомендация Claude — A+B. НЕ выбрано Yar.

## ЛЕСТНИЦА ПОРОГОВ ВОСХОЖДЕНИЯ (Yar, 2026-07-22)

Верхние миры — не просто «этажи», а ПОРОГИ-ИСПЫТАНИЯ. Вертикаль восхождения:

**Метагалактический (Духовный) → Огненный → Синтезный (экополис).**

1. **Порог Духа** (Метагалакт.): клетки-карты несут испытания по структуре 22 ступеней (канон: 22 Великих Аркана Шмакова как «ступени пути духа»). Игроку «выпадает суд» на клетке; ПРОШЁЛ карту-испытание → доступ в Огненный мир.
   - **FIREWALL (approved, COMP-002/SHMAKOV):** «Таро», «Арканы», имена арканов, отсылки к гаданию — ЗАПРЕЩЕНЫ в игровом слое. Игровое имя: **«Пороги Духа»** (слово «Порог» уже живёт в IP: Войны Порогов, Пожиратель Порога — ляжет без шва). Канон Шмакова — только в лоре.
2. **Суд Огня** (Огненный): зал наизнанку, Поядающий огонь, суд ИВО по соответствию (§RESOURCE_MODEL 4c). Прошёл → доступ в Синтезный.
3. **Синтезный экополис** — вершина.

**Два последовательных порога-суда (Пороги Духа → Суд Огня) перед финалом.** Тёмное восхождение через испытания = жанр Dark Cosmic Adventure.

**Canon-подтверждение:** Путь Шмакова активен именно в верхней триаде (Душа→Воля→Огненный) — совпадает с этой лестницей.

**УТОЧНИТЬ (Yar):** «Духовный мир» = Метагалактический ярус (верхняя триада), НЕ отдельный шестой мир? (по умолчанию — да, тот же Духовно-Метагалактический ярус).

## АТТЕСТАЦИИ — КАРТА-СУД НА КАЖДОМ ПЕРЕХОДЕ (Yar, 2026-07-22)

**Принцип:** в каждом мире есть карта-Суд (Аттестация); её прохождение открывает переход выше. Не «дошёл до края → поднялся», а «прошёл аттестацию → мир выше открылся». Суд Огня — частный (грознейший) случай общего механизма.

**Два решения Yar:**
- Аттестации РАЗНЫЕ по характеру (каждый мир проверяет своё), общая рамка «карта-Суд».
- Прохождение КОМАНДНОЕ — мир открывается ВСЕМ разом (анти-quarterback: никто не брошен внизу).

**Условие «команда прошла» (Claude-реком., развилка):** ПОРОГ ГРУППЫ — суммарная величина (чистота/ресурс/решённые испытания) команды ≥ порога яруса. Сильные «доносят» отстающего → стимулирует помощь, не бросание. Альтернативы: все-должны / большинство. ПОДТВЕРДИТЬ Yar.

### ЛЕСТНИЦА АТТЕСТАЦИЙ (4 перехода)

| Переход | Аттестация | Что проверяет (характер) | Провал (командный) |
|---|---|---|---|
| Физический → Тонкий | **Врата Пробуждения** | базовая готовность: команда собрала минимум энергии/присутствий, очистила стартовую Порчу | остаются, копят ещё |
| Тонкий → Метагалакт. | **Пороги Духа** ⚠(=арканы, firewall) | путь познания: сумма Света/решённых загадок-ступеней ≥ порога | не открыт, ищут ещё знание |
| Метагалакт. → Огненный | **Испытание Воли** | готовность к огню: сумма Духа/выдержанных искажений воли ≥ порога | откат, накопление Духа |
| Огненный → Синтезный | **Суд Огня** ⚠ (зал наизнанку, Поядающий) | СООТВЕТСТВИЕ: командная Чистота vs суммарный Груз (§RESOURCE_MODEL 4d, Тройка A) | падение группы на ярус (не смерть) |
| Синтезный → финал | **Допуск экополиса** ⚠ | «достойные сотрудники, не награда»: все пороги пройдены | нет допуска |

Каждая — СВОЯ по содержанию (энергия / знание / воля / чистота / полнота пути), но единый формат «карта-Суд + командный порог». Прописаны детально пока 2 (Пороги Духа, Суд Огня); остальные 3 — слоты, наполнение TBD.

**Жанрово:** нарастающая грозность аттестаций (мягкие Врата внизу → грозный Суд Огня вверху) = кривая Dark Cosmic Adventure (тьма/ставка растёт к вершине, затем катарсис Синтеза).
