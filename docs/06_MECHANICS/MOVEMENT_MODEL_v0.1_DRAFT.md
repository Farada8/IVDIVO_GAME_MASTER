## РЕШЕНИЕ: ЧТО ТАКОЕ ЗОНА
Зона = **сектор кольца** (не абстрактное «место», не только карта). Двойственна:
- часть секторов **напечатана на кольцах** (влезшие на поле);
- остальные — **в колоде карт-секторов**, входят на поле ПО ПРАВИЛУ (замена отработанного), не случайным вбросом.
Поле живёт и меняется, но предсказуемо.

## ПОЛЕ
Концентрические вращающиеся кольца (волвелле / Tzolk'in-style). Каждое кольцо = мир:
Физический (центр-ось) · Тонкий · Метагалактический · Огненный · Синтезный.
Игрок — в ЦЕНТРЕ, неподвижная ось (Логика 1). Кольца вращаются вокруг него.
Нарративное обоснование: «игрок — неподвижный центр, реальность вращается; конъюнкция = вспоминание себя на ярусе». Движение = вращение мира вокруг игрока, НЕ беготня игрока.

## ТРИ ДВИГАТЕЛЯ (по одной функции каждый)

### 1. Кубик — ЕДИНСТВЕННАЯ случайность
Бросок → ход/действия игрока на раунд. Источник напряжения. Только он несёт рандом.

### 2. Кольца — ПРЕДСКАЗУЕМАЯ головоломка (НЕ рулетка)
Вращаются по фиксированным взаимно-простым периодам (2,3,5,7 — канон). Игрок ЗНАЕТ, когда кольца сойдутся в окне конъюнкции → планирует наперёд. Тайминг-головоломка = интеллектуальный слой, противовес кубику.
Вращение инициируется/тактируется определёнными картами, НО по предсказуемым периодам (карта = «тик», не случайный бросок).

### 3. Накал — управляемый подталкиватель
Игрок платит Накал → доворачивает кольцо на 1-2 сектора раньше. Дозированный контроль (ресурс ограничен), БЕЗ второго движения (нет двойной арифметики Л2). Механика роли «Смотритель Колец».

## ИСТОЧНИКИ СЛУЧАЙНОСТИ: РОВНО ОДИН (кубик)
Кольца предсказуемы, замена секторов — по правилу. Обоснование: Costikyan «Uncertainty in Games» — избыток источников случайности убивает ощущение своих решений. Один рандом + мастерство планирования колец = «uncertainty as skill».

## ЗОНА-СЕКТОР: ПОЛЯ КАРТЫ (из запроса Yar, к доработке)

## УРОВНИ СЕКТОРОВ — ЭВОЛЮЦИЯ ПОЛЯ (Yar, 2026-07-22)

Сектор имеет УРОВНИ (базовый → 2…10). Поле не статично — растёт вверх по мере заслуг игроков.

**Триггер подъёма — КОЛЛЕКТИВНАЯ ЗАСЛУГА (не рандом, не таймер):** когда игроки СВОИМИ ЗАСЛУГАМИ переводят мир (или орбиталь) на следующий уровень, открывается КОЛОДА КАРТ этого уровня (2–10) для данного мира/орбитали. Новые сектора-карты **выкладываются ПОВЕРХ старых прямо на поле** (накладка — видно, что мир «поднялся»).

**Три независимых измерения одного сектора (сложившаяся глубина поля):**
| Слой | Что меняет | Триггер |
|---|---|---|
| Сторона (стаб./искаж.) | здоров/заражён | Порча/Разлом |
| **Уровень (1→10)** | **мир поднялся на ступень** | **заслуги игроков (командно)** |
| Вращение | какой сектор напротив | периоды колец |

**Ludonarrative:** механика = тема. Игра про восхождение/преобразование миров → поле буквально растёт по уровням. «Преобразование зоны» как цель = подъём уровня поля (ответ на вопрос, висевший в FORMS_50).

**РИСК ОБЪЁМА (не разрешён):** колоды 2–10 × ~6 сущностей × ~7 секторов ≈ до ~380 карт секторов — неподъёмно для Core Box. МАСШТАБ НЕ ВЫБРАН (Yar: «запиши, масштаб потом»). Кандидаты решения:
- A: Core Box уровни 1–3 картами, выше — допы.
- B: уровни жетонами-накладками (усиливают существующий сектор, не новые карты).
- C: уровни только у якорных секторов (портал/особое).
Рекомендация Claude — A+B. НЕ выбрано Yar.
