# FORMS 50 — Карты Форм v0.1
### Форма = созданная/восстановленная структура, меняющая состояние зоны, портала, Убежища, Корня или закона мира

> **Статус:** DRAFT · Block 6 (Mechanics) · НЕ APPROVED
> **Формат (канон):** каждая Форма = эффект + условие повреждения + условие разрушения. Форма без условий — дыра в балансе.
> **Вид материи:** взят из MATTER_64_TYPES, только ЗОНА A (1–7, общеэзотерич., firewall-нейтральны). Растёт с «высотой» Формы: Физическая внизу → Атмическая наверху. Зона C (Витиическая и др.) НЕ используется на картах (канон: «под заменой»).
> **Цены (`~`):** черновые, в «потенциале». Уточнятся по RESOURCE_MODEL v0.2 (Дух/Свет/Огонь).
> **Firewall:** игровые имена COMMERCIAL; вид материи — тег-мост к канону.
