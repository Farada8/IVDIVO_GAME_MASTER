# CARD_TYPE_SYSTEM v0.1
### Система типов карт

> **Статус:** DRAFT · Block 6 · не APPROVED
> **Метод:** каждый тип проверяется вопросом «отличит ли игрок его от соседнего на столе за секунду?». Тип, который не различается визуально и функционально — не тип.
> **Ограничение реальности:** игрок держит в голове 4–6 типов. Тринадцать — провал обучения.

---

## КРИТЕРИЙ ТИПА

Тип выживает, если:
1. **Отличим** — по рубашке/рамке, за секунду.
2. **Иначе работает** — свой момент розыгрыша или свой формат эффекта.
3. **Не сводится** — другой тип не покрывает его функцию.

**Проваленный критерий = слить или удалить.**

# АУДИТ ТРИНАДЦАТИ ТИПОВ

## 1. ПЕРСОНАЖ

1. **Назначение:** роль игрока. Один глагол, степень, инструменты.
2. **Когда:** выбирается до партии, лежит перед игроком всю партию.
3. **Поля:** имя · глагол (суть) · степень (докуда достаёт) · тип (деятель/держатель/собиратель) · стартовые Импульс/Целостность · **ограничение** (что не делает).
4. **Пример:**
```
┌─ ЧЕЛОВЕК ───────────────────┐
│ Глагол: СОБИРАЕТ СЕБЯ       │
│ Степень: Твердь             │
│ Тип: деятель                │
│ Импульс 3 · Целостность 4   │
│ ─────────────────────────── │
│ Начинаешь ни с чем.         │
│ Можешь взять любой Ключ.    │
│ Ограничение: без Ключа не   │
│ действуешь вне Тверди.      │
└─────────────────────────────┘
```
5. **Стоимость:** ❌ (не разыгрывается).
6. **Ограничение:** один на игрока. Не меняется в партии *(меняется только Перерождением — доп.)*.
7. **Эффект:** задаёт, **что игрок вообще может**.
8. **Контрмера:** ❌ — нельзя «убить персонажа». Целостность 0 = Оглушён на ход.
9. **Тест:** различают ли игроки роли **по глаголу**, а не по силе? *Провал: спрашивают «а кто сильнее?».*
10. **В прототип:** **ДА.** Но все берут Человека — асимметрия не тестируется.

**Вердикт: ОСТАЁТСЯ.** Не карта в колоде — **планшет**.

---

## 2. ЖИТЕЛЬ МИРА

1. **Назначение:** существо/функция мира. Не «монстр» — функция.
2. **Когда:** выкладывается угрозой или сценарием.
3. **Поля:** имя · мир · функция · **сила (может быть переменной!)** · контрмера-подсказка.
4. **Пример:**
```
┌─ ПОЖИРАТЕЛЬ ────── Глубь ───┐
│ Функция: растёт на чужом    │
│ ─────────────────────────── │
│ Сила = Луч игроков в зоне.  │
│ Урон по нему разливает      │
│ порчу: +1 в зону.           │
│ ─────────────────────────── │
│ Слабеет, когда зона пуста.  │
└─────────────────────────────┘
```
5. **Стоимость:** ❌ (не игрок разыгрывает).
6. **Ограничение:** привязан к миру. **Не работает по пустому.**
7. **Эффект:** давление на зону.
8. **Контрмера:** **своя у каждого** — почти никогда урон.
9. **Тест:** понимают ли игроки, что **урон не универсален**? *Провал: бьют всё подряд и фрустрируют.*
10. **В прототип:** **ДА** — Пожиратели, минимум.

**Вердикт: ОСТАЁТСЯ.** Первая карта с переменной силой — параметр берётся **с поля**, не напечатан.

---

## 3. ИНСТРУМЕНТ

1. **Назначение:** чем игрок действует.
2. **Когда:** из руки, в свой ход.
3. **Поля:** имя · мир · стоимость Импульса · эффект · **ограничение**.
4. **Пример:**
```
┌─ КЛИНОК ───────── Твердь ───┐
│ 2 Импульса                  │
│ ─────────────────────────── │
│ Убрать 1 существо из зоны.  │
│ ─────────────────────────── │
│ Не действует на бестелесное.│
│ Против Пожирателя: +1 порча │
│ в зону.                     │
└─────────────────────────────┘
```
5. **Стоимость:** Импульс, иногда + материя.
6. **Ограничение:** **работает только в пределах степени** роли.
7. **Эффект:** по слою.
8. **Контрмера:** Искажение (обращает твой инструмент против тебя).
9. **Тест:** есть ли у игрока выбор, **чем** решать? *Провал: одна карта всегда лучшая.*
10. **В прототип:** **ДА.** Ядро руки.

**Вердикт: ОСТАЁТСЯ.**

---

## 4. ПОРТАЛ

1. **Назначение:** проход между слоями.
2. **Когда:** **открывается сам** при совпадении секторов; или принудительно (сжигание).
3. **Поля:** какие слои соединяет · условие открытия · сколько держится.
4. **Пример:**
```
┌─ ПОРТАЛ ─────────────────────┐
│ Твердь ↔ Высь               │
│ ─────────────────────────── │
│ Открыт, пока секторы кругов │
│ совпадают.                  │
│ Принудительно: сжечь предмет│
│ ─────────────────────────── │
│ Работает в обе стороны.     │
└─────────────────────────────┘
```
5. **Стоимость:** естественно — ❌. Принудительно — предмет.
6. **Ограничение:** **временный.** Вращение уносит совпадение.
7. **Эффект:** доступ к слою.
8. **Контрмера:** Сводчик-босс закрывает узел.
9. **Тест:** планируют ли игроки **под будущее совпадение**? *Провал: не замечают вращения.*
10. **В прототип:** **ДА, но упрощённо** — 1 статичный портал. Вращение — v0.3.

**Вердикт: ОСТАЁТСЯ.** Но не карта — **компонент поля** (сектор круга).

---

## 5. УГРОЗА

1. **Назначение:** ход Глуби. Отдельная колода.
2. **Когда:** Фаза Мира, автоматически.
3. **Поля:** имя · эффект · **условие («не работает, если…»)**.
4. **Пример:**
```
┌─ ГОЛОД ──────── Угроза ─────┐
│ Все Пожиратели идут к зоне  │
│ с наибольшим Лучом игроков. │
│ ─────────────────────────── │
│ Стоимость: 0.               │
│ Не срабатывает, если Луч    │
│ игроков = 0 везде.          │
└─────────────────────────────┘
```
5. **Стоимость:** **0. Всегда.** Структурное преимущество Глуби.
6. **Ограничение:** **не работает по пустому.** Компенсация бесплатности.
7. **Эффект:** давление.
8. **Контрмера:** быть пустым.
9. **Тест:** чувствуют ли игроки, что **их успех кормит угрозу**? *Провал: угроза ощущается случайной.*
10. **В прототип:** **ДА.** 12 карт.

**Вердикт: ОСТАЁТСЯ.** Отдельная колода, отдельная рубашка.

## 6. ПРИЧИНА

1. **Назначение:** истинный источник порчи. **Спрятан.**
2. **Когда:** лежит закрытым с настройки; вскрывается Раскрытием.
3. **Поля:** *(рубашка одинакова у всех!)* · внутри: истинный/ложный · что требуется для выжигания.
4. **Пример:**
```
┌─ [рубашка: ИСТОЧНИК?] ──────┐
│  (вскрывается Раскрытием)   │
├─ ИСТИННЫЙ КОРЕНЬ ───────────┤
│ Порча в этой зоне растёт    │
│ +1 дополнительно.           │
│ ─────────────────────────── │
│ Выжигание: Очищение +       │
│ 2 ресурса. Зона изменена    │
│ навсегда.                   │
└─────────────────────────────┘
```
5. **Стоимость:** вскрытие — 2 Импульса (Раскрытие).
6. **Ограничение:** **раскрытие ≠ решение.** Нашёл — теперь дотянись и заплати.
7. **Эффект:** цель партии.
8. **Контрмера (для угрозы):** Подмена — ложный корень.
9. **Тест:** **держит ли незнание напряжение первые 3 раунда?** *Провал: вскрывают сразу и хоррор кончается.*
10. **В прототип:** **ДА — это ядро хоррора.** 5 жетонов рубашкой вверх.

**Вердикт: ОСТАЁТСЯ.** Единственный тип с **одинаковой рубашкой** — в этом весь смысл.

## 7. ЗАКОН МИРА

1. **Назначение:** правило, действующее весь сценарий.
2. **Когда:** выкладывается при настройке сценария.
3. **Поля:** имя · мир · постоянное правило.
4. **Пример:**
```
┌─ ЗАКОН: ТЯЖЕСТЬ ── Твердь ──┐
│ Весь сценарий:              │
│ Перемещение стоит +1 Импульс│
│ Постройки дают +1 защиты.   │
└─────────────────────────────┘
```
5. **Стоимость:** ❌.
6. **Ограничение:** **не разыгрывается и не убирается** игроками. Меняется только ролью Ману (строит основы) — доп.
7. **Эффект:** задаёт физику сценария.
8. **Контрмера:** ❌ — с законом не спорят, под него подстраиваются.
9. **Тест:** меняет ли закон **стратегию**, а не только числа? *Провал: игроки не замечают.*
10. **В прототип:** **НЕТ.** Один сценарий — законы не нужны. **v0.3.**

**Вердикт: ОСТАЁТСЯ, но в дополнении.** В базе — вариативность сценариев.

## 8. СОБЫТИЕ

1. **Назначение:** разовый сценарный поворот.
2. **Когда:** по триггеру (раунд N, падение зоны).
3. **Поля:** триггер · текст · эффект · выбор.
4. **Пример:**
```
┌─ ГОРОД НЕ УЙДЁТ ── Событие ─┐
│ Триггер: корень вскрыт под  │
│ зоной с населением.         │
│ ─────────────────────────── │
│ Осевшие отказываются уходить│
│ Выжечь: -5 населения.       │
│ Ждать: +1 порча/раунд.      │
│ ─────────────────────────── │
│ Нет правильного варианта.   │
└─────────────────────────────┘
```
5. **Стоимость:** ❌.
6. **Ограничение:** срабатывает **один раз**.
7. **Эффект:** ставит выбор без хорошего варианта.
8. **Контрмера:** ❌ — событие не отменяется.
9. **Тест:** **спорят ли игроки** над выбором? *Провал: очевидный правильный ответ.*
10. **В прототип:** **1 событие** — для проверки, что дилемма работает.

**Вердикт: ОСТАЁТСЯ.**

## 9. ФРАКЦИЯ

1. **Назначение:** сила мира со своим методом.
2. **Когда:** сценарная расстановка.
3. **Поля:** имя · мир · цель · метод · чем помогает · чем опасна.
4. **Пример:**
```
┌─ ОСЕВШИЕ ──────── Твердь ───┐
│ Цель: удержать своё.        │
│ Метод: строят, копят, бьют. │
│ ─────────────────────────── │
│ Помогают: держат зону.      │
│ Опасны: их накопление       │
│ кормит Голодных.            │
└─────────────────────────────┘
```
5. **Стоимость:** ❌.
6. **Ограничение:** фракции **не управляются игроком** — с ними взаимодействуют.
7. **Эффект:** контекст и сделки.
8. **Контрмера:** конфликтующая фракция.
9. **Тест:** ощущаются ли фракции **по-разному**? *Провал: все как «нейтральные NPC».*
10. **В прототип:** **НЕТ.** v0.3.

**Вердикт: ОСТАЁТСЯ, в дополнении.**

## 10. НАСТАВНИК

1. **Назначение:** передаёт способ, не силу.
2. **Когда:** встреча по сценарию.
3. **Поля:** имя · что даёт · **цена (Обет)**.
4. **Пример:**
```
┌─ НАСТАВНИК ──── Духовный ───┐
│ Потрать ход и прими Обет:   │
│ получи Ключ навсегда.       │
│ ─────────────────────────── │
│ Не действует за тебя.       │
│ «Адепт становится —         │
│  его не делают.»            │
└─────────────────────────────┘
```
5. **Стоимость:** ход + Обет.
6. **Ограничение:** **не делает за тебя.**
7. **Эффект:** новый тип действия навсегда.
8. **Контрмера:** давление темпа — некогда учиться.
9. **Тест:** тратят ли игроки ход на обучение при давлении? *Провал: никогда не успевают → тип мёртв.*
10. **В прототип:** **НЕТ.** Требует Обетов (Духовный) — доп.

**Вердикт: ⚠ СЛИТЬ С ЖИТЕЛЕМ.** Наставник — это **режим жителя** (босс/наставник/союзник), а не отдельный тип. У нас в шаблоне жителя уже три режима. Отдельная рубашка не нужна.

## 11. ФОРМА

1. **Назначение:** состояние персонажа на слое.
2. **Когда:** при присутствии на слое / после Перерождения.
3. **Поля:** слой · что можешь · что не можешь.
4. **Пример:**
```
┌─ ФОРМА: В ГЛУБИ ── Глубь ───┐
│ Твой Луч жжёт тебя:         │
│ -1 Целостность за 3 Луча    │
│ в конце хода.               │
│ ─────────────────────────── │
│ Не можешь строить.          │
└─────────────────────────────┘
```
5. **Стоимость:** ❌ — следствие присутствия.
6. **Ограничение:** одна форма на слой.
7. **Эффект:** **разные возможности на разных слоях** — реализация многомировой жизни.
8. **Контрмера:** ❌.
9. **Тест:** чувствует ли игрок, что он **разный** на разных слоях? *Провал: слои ощущаются одинаково.*
10. **В прототип:** **ДА** — 2 формы (Твердь, Глубь). Это и есть проверка гипотезы №1.

**Вердикт: ОСТАЁТСЯ.** Не карта в колоде — **справка на планшете**.

## 12. ОГНЕННОЕ ИСПЫТАНИЕ

1. **Назначение:** порог перехода.
2. **Когда:** попытка подняться.
3. **Поля:** что требует · что даёт · **«отданное не возвращается»**.
4. **Пример:**
```
┌─ ИСПЫТАНИЕ ────── Высь ─────┐
│ Отдай: 3 Луча ИЛИ предмет   │
│ ИЛИ действующий Обет.       │
│ ═══════════════════════════ │
│ ОТДАННОЕ НЕ ВОЗВРАЩАЕТСЯ.   │
│ ═══════════════════════════ │
│ Получи: Ключ Выси.          │
│ Отказ: остаёшься. Урона нет.│
└─────────────────────────────┘
```
5. **Стоимость:** отдать.
6. **Ограничение:** **не бьёт.** Непройденное = остановка.
7. **Эффект:** Ключ/статус.
8. **Контрмера:** прийти пустым.
9. **Тест:** **проигрывают ли из жадности, а не из слабости?** *Провал: платят автоматически, дилеммы нет.*
10. **В прототип:** **ДА** — 2 карты. Проверка гипотезы №3 (Страж).

**Вердикт: ⚠ ЭТО ИНСТРУМЕНТ ВЫСИ.** Формат карт Выси **уже** обязывает вторую строку («отданное не возвращается»). Отдельный тип не нужен — нужна **рамка Выси** внутри типа Инструмент.

**Решение:** визуально выделить (двойная линия), но тип — Инструмент.

## 13. СИНТЕЗНОЕ ДЕЙСТВИЕ

1. **Назначение:** соединение связей.
2. **Когда:** ход игрока, при узле.
3. **Поля:** что соединяет · требует узла · **нет параметра урона**.
4. **Пример:**
```
┌─ СОГЛАСОВАНИЕ ── Синтез ────┐
│ 2 Импульса + свободный узел │
│ ─────────────────────────── │
│ Проверни круг на 1 сектор.  │
│ ─────────────────────────── │
│ Действует на всех, включая  │
│ угрозу.                     │
└─────────────────────────────┘
```
5. **Стоимость:** Импульс + узел.
6. **Ограничение:** **нужны узлы.** Без них пусто.
7. **Эффект:** переконфигурация.
8. **Контрмера:** лишить узлов.
9. **Тест:** ощущается ли соединение **иначе**, чем обычное действие? *Провал: воспринимается как «ещё одна карта».*
10. **В прототип:** **НЕТ.** Синтез — дополнение.

**Вердикт: ⚠ ЭТО ИНСТРУМЕНТ СИНТЕЗА.** Та же логика, что §12: не тип, а **мир внутри типа Инструмент**.

# ⚠ ГЛАВНАЯ НАХОДКА: МИР — ЭТО НЕ ТИП, А ПОЛЕ

Три кандидата (§10, §12, §13) свелись к одному выводу: **Огненное испытание и Синтезное действие — это не типы, а Инструменты разных миров.**

Значит **мир — это поле на карте**, а не тип карты.

| Тип (рубашка) | Мир (рамка/цвет) |
|---|---|
| Инструмент | Глубь · Твердь · Спектр · Духовный · Высь · Синтез |
| Житель | Глубь · Твердь · Спектр · Высь · Синтез |

**Что это даёт:**
- Игрок учит **5 типов** вместо 13.
- Мир читается **цветом рамки** — мгновенно, без чтения.
- Добавление слоя (дополнение) **не добавляет типов** — только новые карты существующего типа. Правила не растут.
- Формат Выси (обязательная вторая строка) — **свойство рамки**, не отдельный тип.

**Это ровно принцип ADR-001:** дополнение повышает разрешение, не меняет систему.

# ПЯТЬ ТИПОВ ДЛЯ ПРОТОТИПА

## Что игрок держит в руках

**1. ИНСТРУМЕНТ** *(рамка = мир)* — 24 карты
Всё, чем игрок действует. Ядро руки.

**2. УГРОЗА** *(чёрная рубашка)* — 12 карт
Отдельная колода. Игрок никогда её не держит.

**3. ЖИТЕЛЬ** *(рамка = мир)* — 6 карт
Выкладывается угрозой. Переменная сила.

**4. ПРИЧИНА** *(единая рубашка)* — 5 жетонов
Спрятанный корень. Вскрывается Раскрытием.

**5. СОБЫТИЕ** — 1 карта
Сценарный поворот с дилеммой.

## Не в колоде, но на столе
- **Персонаж** — планшет (3 шт).
- **Форма** — справка на планшете (2 слоя).
- **Портал** — сектор поля.

## Не в прототипе
Закон мира (v0.3) · Фракция (v0.3) · Наставник (режим Жителя, доп.) · Синтез, Спектр, Духовный (доп.)

# КАК ТЕСТИРОВАТЬ СИСТЕМУ ТИПОВ

| Тест | Что проверяет | Провал |
|---|---|---|
| **Различение** | Игрок отличает тип за секунду, не читая | путает Инструмент с Жителем |
| **Рамка = мир** | Понимает мир по цвету | спрашивает «это какого мира?» |
| **Формат Выси** | Видит, что у карт Выси **всегда две строки** | удивляется цене |
| **Причина** | Не вскрывает всё сразу; **держит напряжение 3 раунда** | вскрывают на 1-м ходу |
| **Переменная сила** | Понимает, что Пожиратель = его Луч | ищет напечатанное число |

**Мета-тест:** объясняются ли все типы **за 3 минуты**? Если нет — типов слишком много.

## ОТКРЫТЫЕ ВОПРОСЫ
1. **Мир как рамка, не тип** — подтвердить. Правит все карточные документы.
2. Причина: 5 жетонов, из них 1 истинный — или несколько истинных?
3. Житель с переменной силой — читается или требует подсчёта каждый раз?
4. Персонаж-планшет vs карта — планшет дороже в печати, но читаемее.
5. Формат Выси (двойная линия) — достаточно визуально или нужен свой цвет?

## GOVERNANCE
- **5 типов в прототипе.** Восемь кандидатов свелись или ушли в дополнения.
- Кубиков нет. Вся неопределённость — рубашка Причины и колода Угрозы.
- Термины на картах — авторские. Источниковые не используются.
