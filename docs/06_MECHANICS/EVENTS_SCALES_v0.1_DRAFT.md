## СТРУКТУРА СОБЫТИЯ

Каждое событие: масштаб · знак · триггер · эффект поля · какую колоду включает · какой фракции адресовано.

**Масштабы (охват, снизу вверх):** Планета → Солнечная система → Галактика → Метагалактика → Ланиакея → Видимая Вселенная. Выше масштаб = шире эффект (планета=1 мир; вселенная=всё поле).

**Знак:**
- ↓ Пониженные возможности — кризис/искажение. Включает колоду «Пониженные» (испытания, дебаффы).
- ↑ Повышенные — благодать/праздник. Включает колоды «Подарки», «Дары Восхождения», «Культивации».

**Триггер:**
- Стихийное — пришло с колоды Событий (не выбирали).
- Решение Совета — Высший Совет ⚠ (посвящённые/аватары) постановляет (напр. «наградить всех жителей»).

**Клановая (фракционная) колода:** событие-веха вкладывает отдельную колоду ДЛЯ ЗАТРОНУТОЙ ФРАКЦИИ (не всем). Асимметрия: разные фракции радуются разным вехам.

## КАК СОБЫТИЕ КАЧАЕТ БАЛАНС ФРАКЦИЙ
↓-события усиливают Голодных/Отражения (беда = их урожай). ↑-события/праздники — Связанным/Сводчикам/всем (восхождение по согласию). Пробирные — на границе (требуют отдать за доступ). Событие = маятник между «просело» и «награждены», игроки борются за то, чтоб качнуть вверх (через Совет).
