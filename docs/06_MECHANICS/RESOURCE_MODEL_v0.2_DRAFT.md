## 1b. ПЯТЬ ВЕЩЕСТВ = ПЯТЬ ЯРУСОВ (Yar)

Каждый мир пирамиды сформирован из своего вещества. Ресурс яруса = вещество этого яруса (не два разных факта — одно). Это РАСШИРЕНИЕ approved-двойки (Свет-вещество + Дух-вещество) до пятёрки.

| Ярус | Вещество | Ресурс | Роль в экономике (гипотеза Claude, TBD Yar) |
|---|---|---|---|
| Физический | **энергия-вещество** | энергия | сырьё, культивируется внизу |
| Тонкий | **свет-вещество** (свет формируется здесь; «не совсем фотоны») | Свет | оборотный |
| Метагалакт. (Духовный) | **дух-вещество** | Дух | оборотный |
| Огненный | **огне-вещество** | Огонь | оборотный |
| Синтезный | **синтез-вещество** | Синтез | вероятно ЦЕЛЬ/финальный подсчёт, не оборотный (как в v1) |

**Связь с approved-каноном:** проект уже фиксировал две core-субстанции — Свет-вещество (сформированная мудрость, образы, фотоны) и Дух-вещество (тоньше атомов, пронизывает воплощения). Здесь двойка расширена до пятёрки, каждое вещество привязано к своему ярусу.

**Родина vs точка получения:** вещество ФОРМИРУЕТСЯ на своём ярусе (свет-вещество — Тонкий, и т.д.), но ПОЛУЧИТЬ карту можно в любом мире (вышестоящее подаёт вниз). Два факта уживаются: родина фиксирована, точка получения любая.

**РАЗВИЛКА (число ресурсов) — 3 или 5?**
- Оборотных играбельных валют, вероятно, ТРИ: Дух / Свет / Огонь.
- Энергия-вещество (физ.) = сырьё внизу (= развилка (а), кандидат: старый Луч).
- Синтез-вещество (синтезн.) = цель/финальный подсчёт, не оборотный трек (согласуется с v1: «Синтез — финальный подсчёт, не трек»).
- Итого: 3 оборотных + 1 сырьё + 1 цель = 5 веществ, РАЗНЫЕ роли. ГИПОТЕЗА Claude, НЕ подтверждено Yar.

## 2. МЕХАНИЗМ ДОБЫЧИ — ЕДИНЫЙ ПАТТЕРН

**Ярус → его обитатели → потенциал.** Ресурс добывается взаимодействием с жителями/иерархиями/орбиталями мира, а не «майнингом клетки».

**ВАЖНАЯ ПОПРАВКА (Yar):** субстанции НЕ заперты каждая на своём ярусе. Игрок может получить ресурсную карту **в ЛЮБОМ мире** — её подаёт **вышестоящий мир / ИВОтец сверху вниз**. Источник вертикальный (сверху), точка получения — где угодно.

Согласуется с approved-механикой «проявление высшего в низшем» (присутствие + дальность + порог) и с нарративным стержнем «игрок уже существует на всех уровнях одновременно».

**ОТКРЫТЫЙ ВОПРОС (влияет на баланс всех ресурсных карт):**
Подача сверху — (а) обусловленная (по стяжанию / выполненному поручению / открытому каналу / посвящению — есть геймплейный затвор) или (б) свободная (в любой момент)? Для Света Yar описывал квесты/поручения → скорее (а). ПОДТВЕРДИТЬ.

**КУЛЬТИВАЦИЯ — В ФИЗИЧЕСКОМ МИРЕ (Yar):** энергия в основном КУЛЬТИВИРУЕТСЯ (добывается/накапливается) в физическом мире — внизу, через труд/действие/квесты. Затем подаётся сверху как Дух/Свет/Огонь. Полная петля: **культивируешь внизу (физ.) → нисходит сверху (ИВОтец) → получаешь в любом мире.** Физический мир (неподвижный центр волвелле) = БАЗА всей экономики, не проходная локация. Защищает от «все бегут наверх»: игрок привязан к физ. миру как к источнику энергии.

**РАЗВИЛКА (влияет на карты) — «энергия» это:**
- (а) СЫРЬЁ: культивируешь базовую энергию внизу → высшие миры возвращают оформленной как Дух/Свет/Огонь (цикл «труд внизу → благодать сверху»). Кандидат на сырьё: старый **Луч** (= базовая энергия физ. мира, наверху дифференцируется в тройку; тогда Луч не выбрасывается, а встаёт в основание). ИЛИ Материя из v0.1.
- (б) ОБЩЕЕ ИМЯ: «энергия» = собирательное для тройки, уточняет лишь МЕСТО добычи (физ. мир), отдельного сырья нет.
Гипотеза Claude: (а). НЕ подтверждено Yar.

## 4. СВЕТ — РАСШИРЕННО (путь познания)

Свет измеряет ДВА параметра: **объём знания** И **умение действовать Светом** (не только копить — применять).

Прогрессия: порции Света → **посвящения** → «знаки» (степени). Открывает прямое взаимодействие с Тонким миром — **Учителями Лучей**.

**«Путь Света» — ОПЦИОНАЛЬНЫЙ сюжетный трек.** Один из главных ролевых сюжетов, но не обязательный: можно проходить игру другими мирами/сюжетами (путь Духа = действие, путь Огня = преображение). → параллельные пути прохождения = реиграбельность. Ложится на асимметрию ролей (Провидец/Толкователь — свет; Страж/Ходок Дна — дух).
