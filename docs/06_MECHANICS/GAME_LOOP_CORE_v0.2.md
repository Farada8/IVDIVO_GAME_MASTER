# GAME_LOOP_CORE v0.2
### IVDIVO Core Box — основа правил (rules foundation)

> **Статус:** DRAFT · PROPOSED · не APPROVED
> **Жанр:** cooperative cosmic adventure / mythic cosmology campaign game
> **Наследует:** GAME_CORE_LOOP_v0.1 (3 видимых слоя, Луч, проявление высшего в низшем, портал-вращение, «победа = выжечь И уцелело»)
> **Область:** Core Box — первая коммерческая коробка. Кооперативная кампания.
> **Тест документа:** можно ли по нему собрать прототип и сыграть петлю? Числа — намеренно не финализированы (см. §14).

## 0. ОДНА ФРАЗА

**Реальность треснула снизу. Вы не убиваете трещину — вы решаете, чем платить за то, чтобы её закрыть.**

## 1. СЛОВАРЬ CORE BOX

Игрок видит эти слова. Канонические/источниковые термины (ИВДИВО, теософия) сюда **не выходят** — firewall.

| Термин Core Box | Что это | Наследие v0.1 / канон |
|---|---|---|
| **Персонаж** | фигура игрока, присутствующая на слоях сразу | «игрок-вертикаль» |
| **Роль** | архетип с одним глаголом действия | 4 роли в базе из 11 |
| **Личная колода** | индивидуальный набор карт роли | рука/добор |
| **Убежище** | общий дом-база группы; куда возвращаются, что защищают | новый узел (см. §2) |
| **Зона мира** | клетка/область на слое | зоны, Осевшие |
| **Портал / Вихрь** | подвижный проход между зонами/слоями | портал-вращение v0.1 §10 |

| **Разрыв** | быстрый силовой способ закрыть — режет, теряет | новый выбор финала |

| **Луч** | единый ресурс, меняющий знак по слою | Луч v0.1 §8 |

## 2. КОМПОНЕНТЫ ПЕТЛИ (Core Box)

- **Персонажи** — по одному на игрока (2–4). Каждый на слоях присутствует сразу.
- **Роли** — 4 в базе (собиратель себя / правящий стихиями / возвращающийся к людям / созидающий из наличного). Каждая — один глагол, ни одна не управляет другими игроками.
- **Личные колоды** — своя у каждой роли; определяют доступные действия и «голос» персонажа.
- **Убежище** — общая база. Три функции: (1) точка восстановления; (2) хранилище того, что группа успела спасти (это и есть «уцелевшее», которое считается в победе); (3) уязвимая ставка — если давление доходит до Убежища, теряется спасённое. Убежище — не крепость, а то, что вы боитесь потерять.
- **Зоны миров** — поле слоёв; часть заражается, часть держится (Осевшие).
- **Порталы / Вихрь** — подвижные проходы (см. §5).
- **Разломы Реальности** — точки прорыва (см. §6).
- **Корень** — скрытая карта-цель сценария (см. §8).

---

## 3. СТАРТ ПАРТИИ

1. **Выбор сценария** кампании (или пролог для первой партии). Сценарий задаёт: карту зон, стартовое давление, скрытый Корень, колоду Разломов.
2. **Раздача ролей.** Каждый берёт роль и её личную колоду, ставит Персонажа в **Убежище**.
3. **Сборка поля.** Выкладываются зоны видимых слоёв (Глубь / Твердь / Высь как рабочие имена слоёв Core Box). Круги-миры ставятся в стартовое положение секторов.
4. **Посев угрозы.** Сценарий отмечает 1–2 стартово заражённые зоны и замешивает Корень в стопку скрытых карт (игроки не знают, который он).
5. **Стартовые руки.** Добор личной колоды до руки. Луч и Импульс — на стартовых значениях.
6. **Первый порядок хода** — по правилу сценария. Партия начинается с **Завязки**: давление видно, источник — нет.

---

## 4. ФАЗЫ РАУНДА

Раунд = ходы всех игроков по часовой → **Фаза Мира** (автоматическая).

**Фаза Мира — 5 шагов (порядок строгий):**

1. **Вращение.** Круги-миры проворачиваются на 1 сектор. Проверить совпадения секторов → открыть/закрыть **Порталы/Вихрь**. Это часы мира: темп без искусственного счётчика.
2. **Прорыв.** Вскрыть верхнюю карту колоды Разломов. Если условие выполнено — открывается/растёт **Разлом Реальности** (см. §6), из него выходит **Проявление**.
3. **Рост порчи.** Каждая заражённая зона усиливается; при пороге зона падает, её порча вливается в трек **Корня**.
4. **Голод / Проявления.** Проявления двигаются к зонам с наибольшим накоплением игроков (и к Убежищу, если оно ближе всего к богатой зоне). Их сила = порча зоны.
5. **Оцепенение.** Каждая обойдённая (незатронутая) зона усиливает блокировку. Единственное, что растёт от **бездействия**.

**Несущее свойство (из v0.1):** угроза не платит за действия, но **не работает по пустому** — ей нужен носитель, накопление, ваша карта. Бедный почти неуязвим; жадный кормит мир.

---

## 5. ХОД ИГРОКА

Пять фаз, строго по порядку (наследие v0.1 §3):

1. **Восстановление** — Импульс до максимума; добор личной колоды; проверка действующих Обетов (не восстанавливаются — держатся).
2. **Присутствие** — объявить, на каком слое действуешь этот ход. Нужна точка присутствия в слое; действия слоя доступны только при Ключе слоя.
3. **Действия** — тратить Импульс (см. §6-действий ниже).
4. **Установка** — разместить одно: Якорь / постройку / Нить / посев.
5. **Передача** — ход следующему.

## 6. ДЕЙСТВИЯ ИГРОКА

Из списка базовых действий Core Box **урон — только одно** (структурный вывод из v0.1: игра выталкивается из боевой парадигмы, а не запрещает бой):

| Действие | Слой | Суть |
|---|---|---|
| **Перемещение** | любой | шаг по зоне/Тропе; вход в Портал |
| **Отсечение** | Твердь | единственный прямой урон в игре |
| **Постройка / Укрепление** | Твердь | усилить зону или Убежище; добавляет Груз |
| **Раскрытие** | Спектр | ищет скрытое (Корень, Искажение, ложный след); делает тебя видимым |
| **Очищение** | Высь | убирает порчу зоны необратимо |
| **Преобразование частей** | Высь/Синтез | переплавляет накопленное в «совершенную часть» |
| **Проявление высшего в низшем** | сверху вниз | центральная механика: 3 условия обязательны (присутствие + вертикальная дальность + порог) |
| **Принять Обет** | Духовный | необратим в партии; тратит Свободу |
| **Соединение** | Синтез | связать узлы; урона нет |

**Свободные действия** (не стоят Импульса): сброс карты, принятие Обета, возврат долга.

## 7. КАК РАБОТАЕТ ПОРТАЛ / ВИХРЬ

Порталы **подвижны** — в этом отличие от статичных проходов в жанровых аналогах.

- **A. Естественный (Вращение).** Круги-миры вращаются с разными периодами. Секторы совпали → Портал **открылся сам**; совпадение прошло → закрылся. Вихрь — визуальный образ этого открытого совпадения.
- **B. Принудительный (Сжигание).** Сжечь предмет/накопление → открыть Портал вне цикла. Дорого, но не нужно ждать совпадения.
- **C. Согласование (роль).** Провернуть круг на сектор вручную — работает на всех, включая угрозу (может открыть проход и Проявлениям).
- **Столп.** Совпадение секторов нескольких кругов разом → сквозной канал через слои. Открывается верхними обитателями (Планетарные Духи — см. FIRE_WORLD, орбиталь 2). Действие всеми слоями одновременно.

**Риск:** Вихрь — двусторонний. Открытый вами проход может выпустить Проявление вам за спину.

## 8. КАК ПОЯВЛЯЕТСЯ РАЗЛОМ РЕАЛЬНОСТИ

Разлом — это когда давление снизу **пробило слой**, а не просто выросло в зоне.

- Возникает на шаге **Прорыв** Фазы Мира, когда: (а) заражённая зона достигла порога падения; или (б) карта колоды Разломов вскрыла условие; или (в) игрок открыл Вихрь туда, где давление уже критично.
- Разлом — **стойкая метка на зоне**: пока открыт, каждый раунд рождает **Проявление** и тянет соседние зоны к заражению.
- Закрыть Разлом можно только **Очищением зоны** или закрытием питающего Портала. Урон по Проявлениям Разлом не закрывает — они следствие.
- **Связь с Корнем:** множащиеся Разломы = дыхание Корня. Чем ближе к Корню, тем чаще Прорывы.

---

## 9. КАК РАСКРЫВАЕТСЯ ИСКАЖЕНИЕ

Искажение — скрытая карта, обращающая **твой** ресурс или действие против тебя. Наследует Метку v0.1: ставит его **только угроза** (колода/сценарий), ни один игрок не может наложить Искажение на другого — поэтому от него никто за столом не выигрывает и runaway не возникает.

- **Наложение.** На шаге Прорыв или через Проявление игрок получает скрытое Искажение. Он **не знает** прямо — только косвенные признаки (карта роли ведёт себя не так, ресурс утекает).
- **Раскрытие.** Действие **Раскрытие** (или роль-функция «Отражения») вскрывает Искажение — переводит его из скрытого в явное.
- **Снятие — цепь из ролей** (не один игрок): обнаружить → снять за цену → разомкнуть. Разные шаги разных ролей — так снятие Искажения кооперативно, а не «сильная роль всё чинит».
- **Пока скрыто** — Искажение периодически выполняет условие в пользу угрозы (например, тихо кормит ближайший Разлом ресурсом жертвы).

---

## 10. КАК ОТКРЫВАЕТСЯ КОРЕНЬ

Корень спрятан среди скрытых карт с начала партии. Его нельзя «найти боем».

- **Раскрытие** в правильной зоне снимает скрытость с карты. Ложный след (**Подмена**) возможен — вскрытая карта может оказаться не Корнем.
- Чтобы Корень стал **уязвим**, нужно выполнить три условия проявления высшего в низшем относительно него: присутствие рядом + вертикальная дальность роли достаёт + набран порог ресурса.
- Открытый Корень **не атакуется** обычным уроном. К нему применяется финальный выбор: **Разрыв** или **Преобразование** (§11–12).
- Пока Корень скрыт, давление растёт; открытие Корня — переход сценария в стадию **Цена**.

---

## 11. КАК РАБОТАЕТ РАЗРЫВ

Разрыв — **быстрый силовой** способ закрыть Корень. Режет узел, которым Корень держится за реальность.

- **Дёшево по времени, дорого по цене.** Закрывает Корень немедленно, но **выжигает вокруг**: соседние зоны, спасённое в Убежище, накопленные части — часть этого сгорает.
- Не требует полного набора условий Преобразования — можно «дожать», когда времени нет.
- **След в кампании:** Разрыв оставляет шрам — выжженную зону, которая в следующих сценариях приходит уже повреждённой (см. §13).
- Разрыв ведёт к **Пирровой** линии исхода: корень закрыт, но мир беднее.

---

## 12. КАК РАБОТАЕТ ПРЕОБРАЗОВАНИЕ

Преобразование — **медленный сохраняющий** способ. Не режет узел, а переплавляет Корень в «совершенную часть».

- **Дорого по подготовке, дёшево по потерям.** Требует полного набора: присутствие, дальность, порог, и достаточно **Луча-Истока** (накопление, проведённое через Высь).
- Ничего вокруг не выжигает; напротив — даёт «совершенную часть», из которой в Синтезном мире собирается поле (см. SYNTHESIS_WORLD: клетки из совершенных частей).
- **Условие-барьер:** нельзя преобразовать, придя «богатым грузом» — Груз наказывается на Пороге (Страж Порога). Преобразование доступно тому, кто отдал лишнее.
- Ведёт к **Полной/Обычной** линии исхода: корень закрыт, ценное уцелело.

**Разрыв vs Преобразование — сердце выбора Core Box:** не «сильно/слабо», а «быстро и с потерями» против «дорого и с сохранением». Оба закрывают Корень. Разными ценами.

---

## 13. КАК ПАРТИЯ ПОБЕЖДАЕТ

Условие (наследие v0.1): **Корень закрыт И что-то ценное уцелело.**

| Исход | Условие | Способ |
|---|---|---|
| **Полная** | Корень закрыт; зоны Осевших живы; Убежище цело; никто не под Искажением | обычно Преобразование |
| **Обычная** | Корень закрыт; часть потеряна | Преобразование или мягкий Разрыв |
| **Пиррова** | Корень закрыт, но выжжено и ценное | обычно Разрыв |

«Закрыт» — недостаточно. **Уцелело** — вот что считается. Пиррова засчитывается как победа сценария, но переходит в кампанию как **утрата**.

---

## 14. КАК ПАРТИЯ ПРОИГРЫВАЕТ

Проигрыш — не «умерли в бою», а **мир поглощён быстрее, чем закрыт Корень**. Триггеры (любой):

- **Падение Убежища** — давление/Проявления дошли до базы и уничтожили спасённое.
- **Каскад Разломов** — число открытых Разломов превысило предел: реальность рвётся быстрее, чем чинится.
- **Часы вышли** — сценарий дошёл до конца трека раундов, а Корень не открыт/не закрыт.
- **Оцепенение** — слишком много зон заблокировано бездействием; поле застыло.

Проигрывают **не от слабости**, а от жадности (кормили Проявления накоплением) и от избегания (Оцепенение). Это и есть тематическое ядро.

---

## 15. КАК ПОСЛЕДСТВИЯ ПЕРЕХОДЯТ В КАМПАНИЮ

Принцип (наследие GAME_CORE_LOOP §14, SYNTHESIS_WORLD): **переносится не сила, а последствия.**

- **Шрамы.** Зона, выжженная Разрывом, входит в следующий сценарий повреждённой (меньше защиты, выше стартовое давление).
- **Спасённое.** То, что уцелело в Убежище, переносится как ресурс/связь — из него в поздней кампании собирается поле Синтеза.
- **Совершенные части.** Каждое Преобразование даёт часть, которая позже становится клеткой пути в Синтезном мире (замыкание вертикали).
- **Долги и Обеты.** Принятые Обеты и незакрытые Искажения могут тянуться в следующий сценарий.
- **Двигатель кампании:** давление снизу не поднимается само — только через открытую кем-то дверь. Кампания — расследование, **кто открыл**: культы (по глупости), конторы (намеренно). Разрыв закрывает симптом; чтобы закрыть причину, кампания идёт вверх по стадиям (Пробуждение → Прозрение → Цена → Отречение → Сборка).

## 16. ОТКРЫТЫЕ ВОПРОСЫ (для v0.3 и чисел)

1. **Числа не заданы намеренно.** Пороги падения зон, лимит каскада Разломов, стоимость Разрыва/Преобразования, длина трека раундов — на плейтест.
2. **Геометрия дисков (DEC-003)** — открытие Порталов/Вихря и Столпа зависит от того, есть ли у верхних миров собственные кольца. Блокер, унаследованный от верхних миров.
3. **Убежище** — новый узел, не было в v0.1. Требует спецификации: вместимость спасённого, условия падения, можно ли переносить/укреплять между сценариями.
4. **Разрыв/Преобразование** относительно ролей — все ли роли могут оба, или роль сужает выбор?
5. **Число ролей в Core Box** — 4 (v0.1) подтвердить для коммерческой коробки (пересечение с DEC-009).
