## 1. Назначение

Этот индекс фиксирует три созданные таблицы карточных данных IVDIVO и их размещение в Google Drive.

Таблицы относятся к раннему тестовому слою игры и НЕ являются финальным rulebook.

| Инструменты | 100 | DRAFT / TO-PLAYTEST | ключи, якоря, сосуды, резонаторы, компасы, печати, зеркала, проводники силы |

## 4. Принципы дизайна

1. Карты должны быть понятными и функциональными.
2. Стартовые карты должны быть слабее карт рынка.
3. Карты развития должны давать выбор, а не просто `+1`.
4. Инструменты не являются только оружием; это средства действия в космологической игре.
5. Все эффекты пока тестовые и требуют бумажного playtest.

## 6. Что требует следующей фиксации

1. Сохранить CSV-версии таблиц в GitHub как raw data, если решено хранить полный датасет не только в Drive.
2. Связать карточные таблицы с `DOCUMENT_REGISTRY.md`.
3. Добавить запись в `CHANGELOG.md`.
4. После тестов создать версии v0.2 с удалением слабых / лишних / слишком сильных карт.
